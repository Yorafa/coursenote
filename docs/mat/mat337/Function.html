Limits and Continuity​

We define f:S→Rmf: S \to \R^mf:S→Rm, where S∈RnS \in \R^nS∈Rn, its limit exists at aaa if ∀ϵ>0,∃δ>0,∥x−a∥<δ  ⟹  ∥f(x)−v∥<ϵ\forall \epsilon > 0, \exists \delta > 0, \|x - a\| <\delta \implies \|f(x) - v\| < \epsilon∀ϵ>0,∃δ>0,∥x−a∥<δ⟹∥f(x)−v∥<ϵ where we write as lim⁡x→af(x)=b\lim\limits_{x\to a} f(x) = bx→alim​f(x)=b

  • If limit exists, we can also say fff is continuous at aaa
