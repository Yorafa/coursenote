Compact​

A subset SSS of Rn\R^nRn is compact if every sequence in SSS has a convergent subsequence to a point in SSS.

  • SSS then is closed and bounded
  • Every closed subset of a compact set is compact
  • [a,b]n[a,b]^n[a,b]n is compact for any a,b∈Ra,b \in \Ra,b∈R

THE HEINE-BOREL THEOREM: a subset SSS of Rn\R^nRn is compact if and only if closed and bounded.
