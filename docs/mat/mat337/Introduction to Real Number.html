Introduction to Real Number

Beyond real number, let's review some about rational number first. Rational number a.k.a the extension of natural number where we want to separate a integer.

Rational number​

That's we define ration number as ∀q∈Q,q=a/b,a,b∈N,b≠0\forall q\in \mathbb{Q}, q = a/b, a,b \in \N, b\ne 0∀q∈Q,q=a/b,a,b∈N,b=0

We would like to use decimal expansion to present rational number. But the interesting thing found here, we have many decimal expansions, how can we found the related one. For the finite decimal expansion, we just simply use 10n10^n10n where nnn is the number of digits after dot to times such expansion then we have a integer, and now we can present it as a/ba/ba/b so that all finite decimal expansions are rational. How about infinite decimal expansion? Some of them are eventually periodic, some of them are not. Among those expansion, how can we judge if it's a rational number?

Let's define the eventually periodic first. An infinite decimal x=a0.a1a2⋯x = a0.a1a2\cdotsx=a0.a1a2⋯ is eventually periodic if there are positive integers nnn and kkk such that ai+k=aia_{i+k} = a_iai+k​=ai​ for all i>ni > ni>n.

  • let x=a0.a1a2…x = a0.a1a2\ldotsx=a0.a1a2… be arbitrary eventually periodic infinite decimal where ∃n,k∈N,n≥0,∀i∈N,i>n  ⟹  ai+k=ai\exists n,k\in \N, n \ge 0, \forall i \in \N, i > n \implies a_{i+k} = a_i∃n,k∈N,n≥0,∀i∈N,i>n⟹ai+k​=ai​
  • Since 10nx=a0⋯an.an+1…10^{n} x = a_0\cdots a_n.a_{n+1}\ldots10nx=a0​⋯an​.an+1​…, 10n+kx=a0⋯an⋯an+k.an+k+1…10^{n+k} x = a_0\cdots a_n\cdots a_{n+k}. a_{n+k+1} \ldots10n+kx=a0​⋯an​⋯an+k​.an+k+1​… and the decimals are the same (where n+1>nn+1 > nn+1>n and ai=ai+k,∀i>na_i = a_{i+k}, \forall i > nai​=ai+k​,∀i>n) so that we will get a integer for 10n+kx−10nx10^{n+k} x - 10^{n} x10n+kx−10nx, and denote such integer as zzz.
  • x(10n+k−10n)=z  ⟹  x=z10n+k−10nx(10^{n+k} - 10^{n}) = z \implies x = \frac{z}{10^{n+k} - 10^{n}}x(10n+k−10n)=z⟹x=10n+k−10nz​ where z∈Nz\in \Nz∈N and 10n+k−10n∈N10^{n+k} - 10^{n} \in \N10n+k−10n∈N so that xxx is a rational number.

As we can see the infinite periodic decimal expansion can be present as a/b,a,b∈N,b≠0a/b, a,b\in \N, b\ne 0a/b,a,b∈N,b=0, but we can't find a way to present the other that's why we have real number. Sometimes, we also said R\RR is the extension of Q\mathbb{Q}Q

Least Upper Bound Principle (& Greatest Lower Bound)​

First, let's take R\RR to describe some properties.

Let SSS be a subset of real number S⊂RS\subset\RS⊂R be arbitrary

  • ∃M∈R,∀s∈S,M≥s  ⟹  M\exists M\in \R, \forall s\in S, M \ge s \implies M∃M∈R,∀s∈S,M≥s⟹M is an upper bound for SSS.
  • ∃M∈R,∀s∈S,M≤s  ⟹  M\exists M\in \R, \forall s\in S, M \le s \implies M∃M∈R,∀s∈S,M≤s⟹M is an lower bound for SSS
  • ∃L,∀M∈R,s.t.∀s∈S,M≥s,L≥s∧L≤M  ⟺  L\exists L, \forall M \in R, s.t. \forall s\in S, M \ge s, L\ge s \land L \le M \iff L∃L,∀M∈R,s.t.∀s∈S,M≥s,L≥s∧L≤M⟺L is the supremum (least upper bound) for SSS. Denote sup⁡S=L\sup S = LsupS=L
  • ∃L,∀M∈R,s.t.∀s∈S,M≤s,L≤s∧L≥M  ⟺  L\exists L, \forall M \in R, s.t. \forall s\in S, M \le s, L\le s \land L \ge M \iff L∃L,∀M∈R,s.t.∀s∈S,M≤s,L≤s∧L≥M⟺L is the infimum (greatest lower bound) for SSS. Denote inf⁡S=L\inf S = LinfS=L.
  • For ∅,inf⁡∅=+∞,sup⁡∅=−∞\emptyset, \inf \emptyset = + \infty, \sup \emptyset = - \infty∅,inf∅=+∞,sup∅=−∞
  • When we say a set is bounded   ⟺  \iff⟺ the set is both bounded above and bounded below.
  • Denote the maximum of SSS as L, that's max⁡(S)=L\max(S) = Lmax(S)=L. max⁡(S)=L  ⟹  sup⁡S=L\max(S) = L \implies \sup S = Lmax(S)=L⟹supS=L. Similarly, Denote the minimum of SSS as L, that's min⁡(S)=L\min(S) = Lmin(S)=L. min⁡(S)=L  ⟹  inf⁡S=L\min(S) = L \implies \inf S = Lmin(S)=L⟹infS=L.
    • Notice, sup⁡S=L\sup S = LsupS=L does not imply such maximum exists. Similarly for infimum

Axios of Completeness: Every nonempty subset SSS of R\RR that is bounded above has a supremum. Similarly, every nonempty subset SSS of R\RR that is bounded below has an infimum.

Existence of R\RR​

Archimedean Property: (proved by Axios of Completeness)

  1. ∀x∈R,∃n∈N,n>x\forall x\in \R, \exists n\in \N, n > x∀x∈R,∃n∈N,n>x
  2. ∀y>0∈R,∃n∈N,1n<y\forall y > 0 \in \R, \exists n\in \N, \frac{1}{n} < y∀y>0∈R,∃n∈N,n1​<y

It seems all above already use R\RR to present, it happends due to we can actually use those to all subset of R\RR. Back to Q\mathbb{Q}Q, now we have following consequence of completeness:

Density of Q\mathbb{Q}Q in R\RR: ∀x,y∈R,s.t.x<y,∃r∈Q,s.t.x<r<y\forall x,y \in \R, s.t. x < y, \exists r \in \mathbb{Q}, s.t. x < r < y∀x,y∈R,s.t.x<y,∃r∈Q,s.t.x<r<y.

And furthermore, we can write:

  1. ∀a∈R,a=sup⁡{r∈Q:r<a}\forall a\in \R, a = \sup \{r \in \mathbb{Q}: r < a\}∀a∈R,a=sup{r∈Q:r<a}
  2. R={sup⁡(A):A⊆Q,A has upper bound in Q}\R = \{\sup(A): A \subseteq \mathbb{Q}, A \text{ has upper bound in } \mathbb{Q} \}R={sup(A):A⊆Q,A has upper bound in Q}
  3. N⊂Z⊂Q⊂R\N \subset \Z \subset \mathbb{Q} \subset \RN⊂Z⊂Q⊂R

All above show the existance of R\RR is based on N,Z,Q\N, \Z, \mathbb{Q}N,Z,Q by axiom of completeness, we need to know some definition before look at the proof.

We define an ordered pair (A,B)(A, B)(A,B) of nonempty subsets of Q\mathbb{Q}Q as a Dedekind cut with properties:

  1. y∈A,x<y  ⟹  x∈Ay \in A, x < y \implies x \in Ay∈A,x<y⟹x∈A
  2. y∈B,x>y  ⟹  x∈By \in B, x > y \implies x \in By∈B,x>y⟹x∈B
  3. ∀x∈A,∀y∈B,x<y\forall x \in A, \forall y \in B, x < y∀x∈A,∀y∈B,x<y

Now look at the proof:

  1. Let a∈Ra \in \Ra∈R, let Aa={r∈Q:r<a}⊆QA_a = \{r \in \mathbb{Q}: r < a\} \subseteq \mathbb{Q}Aa​={r∈Q:r<a}⊆Q is bounded by Q\mathbb{Q}Q, so that sup⁡(Aa)=a\sup(A_a) = asup(Aa​)=a.
  2. Let b=sup⁡(Aa)b = \sup(A_a)b=sup(Aa​) by Axiom of Completeness, we have b∈Rb \in \Rb∈R.
  3. Assume a≠ba \ne ba=b, then a<ba < ba<b
    1. Case 1: a<ba < ba<b: since b=sup⁡(Aa)b = \sup(A_a)b=sup(Aa​), then b≤ab \le ab≤a which is a contradiction.
    2. Case 2: a>ba > ba>b: ∃r∈Q,b<r<a\exists r \in \mathbb{Q}, b < r < a∃r∈Q,b<r<a by archimedean property, but b=sup⁡(Aa)b = \sup(A_a)b=sup(Aa​) which is a contradiction.
  4. Thus, a=ba = ba=b, so that a∈Ra \in \Ra∈R.

Cardinality​

We use the term cardinality to denote the number of elements in a set. If set AAA has the same number of elements as set BBB, we say AAA and BBB have the same cardinality. We denote A∼BA \sim BA∼B.

Formally, we define the cardinality of a set A∼BA \sim BA∼B if there is a bijection function between AAA and BBB.

Cantor Theorem: Q≁R\mathbb{Q} \not\sim \mathbb{R}Q∼R and N∼Z∼Q\N \sim \Z \sim \mathbb{Q}N∼Z∼Q.

  • can prove by Schroeder-Bernstein Theorem
  • N2∼N\N^2 \sim \NN2∼N then we can recursively apply this.

We define a set is finite if ∃n∈N,s.t.X∼{1,2,…,n}\exists n\in \N, s.t. X \sim \{1,2,\dots,n\}∃n∈N,s.t.X∼{1,2,…,n}. But when we say countbale doesn't mean finite. We define a set is countable if XXX is infinite and X∼NX \sim \NX∼N. And a set XXX is uncountable if XXX is infinite and X≁NX \not\sim \NX∼N.

Schroeder-Bernstein Theorem: X∼Y  ⟺  ∃f:X→1-1Y,∃g:Y→1-1XX \sim Y \iff \exists f: X \overset{\text{1-1}}\to Y, \exists g: Y \overset{\text{1-1}}\to XX∼Y⟺∃f:X→1-1Y,∃g:Y→1-1X

  • A sequence of countable sets is countable.

Since real number is a big set, we may find some special number in usage. We define r∈Rr \in \Rr∈R is algebraic if rrr is a root of a polynomial with integer coefficients. And we can find that the set of algebraic number is countable. The set of non-algebraic number is dense in R\RR.
