Norms

In the previous section, we study the Euclidean norm, but how about the general norm?

Let VVV be vector space over R\RR (not exactly R\RR, is fine for Rm\R^mRm or polynomail space etc), we define the norm on VVV is a function ∥⋅∥\|\cdot\|∥⋅∥ on VVV taking values in [0,∞)[0, \infty)[0,∞) satisfied the following properties:

  • ∀x∈V,x=0  ⟹  ∥⋅∥=0\forall x \in V, x = 0 \implies \|\cdot\| = 0∀x∈V,x=0⟹∥⋅∥=0 (positive definite)
  • ∀x∈V,∥αx∥=∣α∣∥x∥\forall x \in V, \|\alpha x\| = |\alpha| \|x\|∀x∈V,∥αx∥=∣α∣∥x∥ (homogeneous)
  • ∀x,y∈V,∥αx+βy∥≤∣α∣∥x∥+∣β∣∥y∥\forall x, y \in V, \|\alpha x + \beta y\| \leq |\alpha| \|x\| + |\beta| \|y\|∀x,y∈V,∥αx+βy∥≤∣α∣∥x∥+∣β∣∥y∥ (triangle inequality)

And we also define (V,∥⋅∥)(V, \|\cdot\|)(V,∥⋅∥) is a normed vector space, and we define the unit ball over such space is the set {x∈V:∥x∥≤1}\{x \in V: \|x\| \leq 1\}{x∈V:∥x∥≤1}.

Generalized Convergent​

In a normed vector space (V,∥⋅∥)(V, \|\cdot\|)(V,∥⋅∥), a sequence {vn}\{v_n\}{vn​} is said to be converges to v∈Vv \in Vv∈V if lim⁡n→∞∥vn−v∥=0\lim_{n \to \infty} \|v_n - v\| = 0limn→∞​∥vn​−v∥=0.

A normed vector space is complete if every Cauchy sequence converges to some point in the space, we also call it Banach space.

And a subset CCC of VVV is closed if V−CV - CV−C is open, or equivalently, it contains all its limit points, or (vn)∈C,(vn)→v  ⟹  v∈C(v_n) \in C, (v_n) \to v \implies v \in C(vn​)∈C,(vn​)→v⟹v∈C.

A sequence (vn)(v_n)(vn​) in a normed vector space VVV converges to a vector vvv if and only if for each open set UUU containing vvv, there is an NNN such that vn∈Uv_n \in Uvn​∈U for all n≥Nn \geq Nn≥N.

Inner Product Space​

Recall inner product on a vector space VVV is a function <x,y><x, y><x,y> on pairs (x,y)(x, y)(x,y) of vectors in V×VV \times VV×V taking values in R\RR satisfying the following properties:

  • ∀x∈V,<x,x>≥0\forall x \in V, <x, x> \geq 0∀x∈V,<x,x>≥0 and <x,x>=0  ⟺  x=0<x, x> = 0 \iff x = 0<x,x>=0⟺x=0 (positive definite)
  • ∀x,y∈V,<x,y>=<y,x>\forall x,y \in V, <x, y> = <y, x>∀x,y∈V,<x,y>=<y,x> (symmetric)
  • ∀x,y,z∈V,α,β∈R,<αx+βy,z>=α<x,z>+β<y,z>\forall x,y,z \in V, \alpha, \beta \in \R, <\alpha x + \beta y, z> = \alpha <x, z> + \beta <y, z>∀x,y,z∈V,α,β∈R,<αx+βy,z>=α<x,z>+β<y,z> (bilinearity)

Similarly, an inner product space is a vector space with an inner product.

Some examples of inner product spaces:

  • ℓ2\ell^2ℓ2 consists of all sequences x=(xn)i=1∞x = (x_n)_{i = 1}^{\infty}x=(xn​)i=1∞​ such that ∥x∥2:=∑i=1∞xi2<∞\|x\|_2 := \sqrt{\sum_{i=1}^{\infty} x_i^2} < \infty∥x∥2​:=∑i=1∞​xi2​​<∞ with inner product <x,y>=∑i=1∞xiyi<x, y> = \sum_{i=1}^{\infty} x_i y_i<x,y>=∑i=1∞​xi​yi​. Since its finite, then it is a complete inner product space.

Cauchy-Schwarz inequality: Let VVV be an inner product space, ∀x,y∈V,∣<x,y>∣≤∥x∥∥y∥\forall x, y \in V, |<x, y>| \leq \|x\| \|y\|∀x,y∈V,∣<x,y>∣≤∥x∥∥y∥.

  • if x,yx,yx,y are collinear, then ∣<x,y>∣=∥x∥∥y∥|<x, y>| = \|x\| \|y\|∣<x,y>∣=∥x∥∥y∥. (collinear basically means xxx and yyy are in the same line)
  • ∀f,g∈C[a,b]\forall f, g\in C[a, b]∀f,g∈C[a,b], we have ∣<f,g>∣≤(∫ab∣f(x)∣2dx)1/2(∫ab∣g(x)∣2dx)1/2|<f, g>| \leq (\int_a^b |f(x)|^2 dx)^{1/2} (\int_a^b |g(x)|^2 dx)^{1/2}∣<f,g>∣≤(∫ab​∣f(x)∣2dx)1/2(∫ab​∣g(x)∣2dx)1/2.
