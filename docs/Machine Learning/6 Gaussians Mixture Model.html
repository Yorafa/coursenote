Mixture of Gaussians (or Gaussian Mixture Model (GMM))

We use GMM when the situation that Gaussian latent variable model p(x)=∑zp(x,z)p(x) = \sum_z p(x, z)p(x)=∑z​p(x,z) used for clustering.

A K multivariate Gaussian mixture densities has the form: p(x)=∑k=1KπkN(x∣μk,Σk)p(x) = \sum_{k = 1}^K \pi_k \mathcal{N}(x|\mu_k, \Sigma_k)p(x)=∑k=1K​πk​N(x∣μk​,Σk​) where ∑k=1Kπk=1,∀k,πk≥0\sum_{k = 1}^K\pi_k = 1, \forall k, \pi_k \ge 0∑k=1K​πk​=1,∀k,πk​≥0 is called mixing coefficients.

Consider a latent variable z∈{1,…,K}z \in \{1, \dots, K\}z∈{1,…,K}, then p(z=k)=πkp(z = k) = \pi_kp(z=k)=πk​ and p(x∣z=k)=N(x∣μk,Σk)p(x|z = k) = \mathcal{N}(x|\mu_k, \Sigma_k)p(x∣z=k)=N(x∣μk​,Σk​) with joint density p(x,z=k)=p(z=k)p(x∣z=k)=πkN(x∣μk,Σk)p(x, z = k) = p(z = k)p(x|z = k) = \pi_k \mathcal{N}(x|\mu_k, \Sigma_k)p(x,z=k)=p(z=k)p(x∣z=k)=πk​N(x∣μk​,Σk​).

That is, we can also have p(z=k∣x)=πkN(x∣μk,Σk)∑k=1KπkN(x∣μk,Σk)p(z = k|x) = \frac{\pi_k \mathcal{N}(x|\mu_k, \Sigma_k)}{\sum_{k = 1}^K \pi_k \mathcal{N}(x|\mu_k, \Sigma_k)}p(z=k∣x)=∑k=1K​πk​N(x∣μk​,Σk​)πk​N(x∣μk​,Σk​)​.

Since multivariate Gaussian, let denote {x1,…,xn}=X∈Rn×m\{x_1, \ldots, x_n\} = X \in \R^{n \times m}{x1​,…,xn​}=X∈Rn×m and z∈Rnz\in \R^nz∈Rn. Then we have the likelihood function log⁡p(X∣π,μ,Σ)=∑i=1nlog⁡∑k=1KπkN(xi∣μk,Σk)\log p(X|\pi, \mu, \Sigma) = \sum_{i = 1}^n \log \sum_{k = 1}^K \pi_k \mathcal{N}(x_i|\mu_k, \Sigma_k)logp(X∣π,μ,Σ)=∑i=1n​log∑k=1K​πk​N(xi​∣μk​,Σk​).

Then we have the MLE μk=∑inp(z=k∣xi)Nkxi\mu_k = \sum_{i}^{n} \frac{p(z = k|x_i)}{\mathcal{N}_k} x_iμk​=∑in​Nk​p(z=k∣xi​)​xi​, Nk=∑inp(z=k∣xi)\mathcal{N}_k = \sum_{i}^{n} p(z = k|x_i)Nk​=∑in​p(z=k∣xi​), Sigmak=∑inp(z=k∣xi)Nk(xi−μk)(xi−μk)TSigma_k = \sum_{i}^{n} \frac{p(z = k|x_i)}{\mathcal{N}_k} (x_i - \mu_k)(x_i - \mu_k)^TSigmak​=∑in​Nk​p(z=k∣xi​)​(xi​−μk​)(xi​−μk​)T, πk=Nkn\pi_k = \frac{\mathcal{N}_k}{n}πk​=nNk​​.

EM Algorithm​

Since MLE does not have a closed form solution and the estimation depend on the posterior distribution p(z=k∣xi)p(z = k|x_i)p(z=k∣xi​). We can use EM algorithm to estimate the parameters.

The steps are:

  • Initialize π,μ,Σ\pi, \mu, \Sigmaπ,μ,Σ.
  • E-step: for each k,ik, ik,i compute the posterior p(z=k∣xi)=πkN(xi∣μk,Σk)∑k=1KπkN(xi∣μk,Σk)p(z = k|x_i) = \frac{\pi_k \mathcal{N}(x_i|\mu_k, \Sigma_k)}{\sum_{k = 1}^K \pi_k \mathcal{N}(x_i|\mu_k, \Sigma_k)}p(z=k∣xi​)=∑k=1K​πk​N(xi​∣μk​,Σk​)πk​N(xi​∣μk​,Σk​)​.
  • M-step: re-estimate π,μ,Σ\pi, \mu, \Sigmaπ,μ,Σ using the mle formula.
  • Evaluate the log-likelihood and check for convergence.

More general one: Consider a general setting with latent variables. i.e. Observed dataset X∈RN×DX\in \R^{N\times D}X∈RN×D, latent variables Z∈RN×KZ \in \R^{N×K}Z∈RN×K. Maximize the log-likelihood log⁡p(X∣θ)=log⁡(∑Zp(X,Z∣θ))\log p(X|\theta) = \log (\sum_Z p(X, Z|\theta))logp(X∣θ)=log(∑Z​p(X,Z∣θ)):

  • Initialize parameters θold\theta_{old}θold​.
  • E-step: use θold\theta_{old}θold​ to compute the posterior p(Z∣X,θold)p(Z|X, \theta_{old})p(Z∣X,θold​).
  • M-step: θnew=arg max⁡θQ(θ,θold)\theta_{new} = \argmax_{\theta} Q(\theta, \theta_{old})θnew​=argmaxθ​Q(θ,θold​), where Q(θ,θold)=∑Zp(Z∣X,θold)log⁡p(X,Z∣θ)=E[log⁡p(X,Z∣θ)∣X,θold]Q(\theta, \theta_{old}) = \sum_Z p(Z|X, \theta_{old})\log p(X, Z|\theta) = E[\log p(X, Z|\theta) | X, \theta_{old}]Q(θ,θold​)=∑Z​p(Z∣X,θold​)logp(X,Z∣θ)=E[logp(X,Z∣θ)∣X,θold​] which is tractable in many applications.
  • Replace θold←θnew\theta_{old} \leftarrow \theta_{new}θold​←θnew​. Repeat until convergence.

If zzz was observed, then the MLE would be trivial

log⁡p(X,Z∣θ)=∑i=1nlog⁡p(xi,zi∣θ)=∑i=1n∑k=1KI(zi=k)log⁡(πkN(xi∣μk,Σk))\log p(X, Z|\theta) = \sum_{i = 1}^n \log p(x_i, z_i|\theta) = \sum_{i = 1}^n\sum_{k=1}^K\mathbb{I}(z_i = k) \log (\pi_k \mathcal{N} (x_i|\mu_k, \Sigma_k))logp(X,Z∣θ)=∑i=1n​logp(xi​,zi​∣θ)=∑i=1n​∑k=1K​I(zi​=k)log(πk​N(xi​∣μk​,Σk​))

For the E-step: p(Z∣X,θ)=∑i=1np(zi∣X,θ)p(Z|X, \theta) = \sum_{i = 1}^n p(z_i|X, \theta)p(Z∣X,θ)=∑i=1n​p(zi​∣X,θ) we have p(zi=k∣X,θ)=p(zi=k∣xi,θ)=πkNm(xi∣μk,Σk)∑j=1KπjNm(xi∣μj,Σj)p(z_i = k|X, \theta) = p(z_i = k|x_i, \theta) = \frac{\pi_k \mathcal{N}_m(x_i|\mu_k, \Sigma_k)} {\sum_{j=1}^K \pi_j \mathcal{N}_m(x_i|\mu_j , \Sigma_j)}p(zi​=k∣X,θ)=p(zi​=k∣xi​,θ)=∑j=1K​πj​Nm​(xi​∣μj​,Σj​)πk​Nm​(xi​∣μk​,Σk​)​.

For the M-step: E(I(zi=k)∣X,θold)=p(zi=k∣X,θold)E(\mathbb{I}(z_i = k)|X, \theta_{old}) = p(z_i = k|X, \theta_{old})E(I(zi​=k)∣X,θold​)=p(zi​=k∣X,θold​) and so E[log⁡p(X,Z∣θ)∣X,θold]=∑i=1n∑k=1Kp(zi=k∣X,θold)log⁡(πkN(xi∣μk,Σk))E[\log p(X, Z|\theta) | X, \theta_{old}] = \sum_{i = 1}^n\sum_{k=1}^K p(z_i = k|X, \theta_{old})\log (\pi_k \mathcal{N}(x_i|\mu_k, \Sigma_k))E[logp(X,Z∣θ)∣X,θold​]=∑i=1n​∑k=1K​p(zi​=k∣X,θold​)log(πk​N(xi​∣μk​,Σk​)).
