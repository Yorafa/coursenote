Kernal Method

Define ψ(x):RD→RM\psi(x): \R^D \to \R^Mψ(x):RD→RM and input data X∈RN×DX \in \R^{N \times D}X∈RN×D and Ψ∈RN×M\Psi \in \R^{N \times M}Ψ∈RN×M, where Ψ=ψ(X)\Psi = \psi(X)Ψ=ψ(X). Then we have the prediction matrix Y^=Ψw\hat{Y} = \Psi wY^=Ψw. Then y∣x∼N(wTψ(x),σ2)y|x \sim N(w^T\psi(x), \sigma^2)y∣x∼N(wTψ(x),σ2).

if N<MN < MN<M, then MLE will not be unique. Flexibility may require a large number MMM of features which may need to depend on NNN. That is, we need kernal method to solve this problem (which is more flexible).

Recall ridge regression problem, we need to minimize E(w)=0.5∥y−Ψw∥2+λ2wTwE(w) = 0.5 \|y - \Psi w\|^2 + \frac{\lambda}{2}w^T wE(w)=0.5∥y−Ψw∥2+2λ​wTw (i.e. ΔE(w)=ΨTΨw−ΨTy+λw=0\Delta E(w) = \Psi^T \Psi w - \Psi^T y + \lambda w = 0ΔE(w)=ΨTΨw−ΨTy+λw=0) and we have solution w=1λΨT(y−Ψw)∈RMw = \frac{1}{\lambda} \Psi^T(y - \Psi w) \in \R^Mw=λ1​ΨT(y−Ψw)∈RM.

Now let's look kernal ridge regression, def: gram matrix (Kernal Matrix) K=ΨTΨ∈RM×MK = \Psi^T \Psi \in \R^{M \times M}K=ΨTΨ∈RM×M, where Kij=ψ(x(i))⊤ψ(x(j))=:k(x(i),x(j))K_{ij} = \psi(x^{(i)})^{\top} \psi(x^{(j)}) =: k(x^{(i)}, x^{(j)})Kij​=ψ(x(i))⊤ψ(x(j))=:k(x(i),x(j)) which is is PSD.

And we define a=(y−Ψw)/λ∈RNa=(y-\Psi w)/\lambda \in \R^Na=(y−Ψw)/λ∈RN, then we have w=ΨTaw = \Psi^T aw=ΨTa and E(a)=12∥y−ΨΨTa∥2+λ2aTΨΨTa=12∥y−Ka∥2+λ2aTKaE(a) = \frac{1}{2} \|y - \Psi\Psi^T a\|^2 + \frac{\lambda}{2} a^T \Psi\Psi^T a = \frac{1}{2} \|y - Ka\|^2 + \frac{\lambda}{2} a^T K aE(a)=21​∥y−ΨΨTa∥2+2λ​aTΨΨTa=21​∥y−Ka∥2+2λ​aTKa

And reversely, plug in w=ΨTaw = \Psi^T aw=ΨTa into a=(K+λIN)−1ya = (K + \lambda I_N)^{-1} ya=(K+λIN​)−1y, we have w=ΨT(K+λI)−1yw = \Psi^T (K + \lambda I)^{-1} yw=ΨT(K+λI)−1y.

y^(x)=ψ(x)Tw=ψ(x)TΨTa=k(x)T(K+λI)−1y\hat{y} (x) = \psi(x)^{T} w = \psi(x)^T \Psi^T a = k(x)^T (K + \lambda I)^{-1} yy^​(x)=ψ(x)Tw=ψ(x)TΨTa=k(x)T(K+λI)−1y.
