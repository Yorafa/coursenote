Hidden Markov Model

In previous courses or even previous lecture, we always generally assume data was i.i.d. for convenience purpose, however this may be a poor assumption. Many real life problems are not i.i.d. instead they are sequential data. That is, we make the simplifying assumption that our data can be modeled as a first-order Markov chain p(xt∣x1:t−1)=p(xt∣xt−1)p(x_t|x_{1:t-1}) = p(x_t|x_{t-1})p(xt​∣x1:t−1​)=p(xt​∣xt−1​).

Since the assumption is still restrictive where the states of our variables are fully observed. To solve this problem, we can use Hidden Markov Model (HMM). HMMs hide the temporal dependence by keeping it in the unobserved state (No assumptions on the temporal dependence of observations is made). And then for each observation xtx_txt​, we have a corresponding unobserved hidden/latent variable ztz_tzt​.

  • we have the joint distribution p(x1:T,z1:T)=p(z1)∏t=2Tp(zt∣zt−1)∏t=1Tp(xt∣zt)p(x_{1:T},z_{1:T}) = p(z_1)\prod_{t=2}^T p(z_t|z_{t-1})\prod_{t=1}^T p(x_t|z_t)p(x1:T​,z1:T​)=p(z1​)∏t=2T​p(zt​∣zt−1​)∏t=1T​p(xt​∣zt​).

HMM observations are not limited by any order of Markov chain. For a homogeneous HMM, we have the following:

  • Initial Distribution: π(i)=p(z1=i)\pi(i) = p(z_1 = i)π(i)=p(z1​=i). The probability of the first hidden variable being in state iii.
  • Transition Distribution: Ψ(i,j)=p(zt+1=j∣zt=i),i∈{1,…,k}\Psi(i,j) = p(z_{t + 1} = j|z_t = i), i \in \{1, \ldots, k\}Ψ(i,j)=p(zt+1​=j∣zt​=i),i∈{1,…,k}. The probability of moving from hidden state iii to hidden state jjj.
  • Emission Distribution: ψt(i)=p(xt∣zt=i)\psi_t(i) = p(x_t|z_t = i)ψt​(i)=p(xt​∣zt​=i). The probability of an observed random variable xxx given the state of the hidden variable that "emitted" it.

Forward-Backward Algorithm​

In order to compute the probability of a latent sequence given an observation sequence. (i.e. p(z1:t∣x1:t)p(z_{1:t}|x_{1:t})p(z1:t​∣x1:t​)). We can use the Forward-Backward Algorithm to solve such task.

This taks of hidden state inference breaks down into serveral subtasks:

  • Filtering: compute the posterior over current hidden state, p(zt∣x1:t)p(z_t|x_{1:t})p(zt​∣x1:t​).
  • Prediction: compute the posterior over future hidden state, p(zt+k∣x1:t)p(z_{t+k}|x_{1:t})p(zt+k​∣x1:t​).
  • Smoothing: compute the posterior over past hidden state, p(zk∣x1:t),1<k<tp(z_{k}|x_{1:t}), 1 < k < tp(zk​∣x1:t​),1<k<t.

We can use Forward Recursion p(zt∣x1:t)p(z_t|x_{1:t})p(zt​∣x1:t​) and Backward Recursion p(x1+t:T∣zt)p(x_{1+t:T}|z_t)p(x1+t:T​∣zt​). That is, γt=p(zt∣x1:T)∝p(zt,x1:T)=p(zt,x1:t)p(xt+1:T∣zt,x1:t)=p(zt,x1:t)p(xt+1:T∣zt)∝αt(i)βt(i)\gamma_t = p(z_t|x_{1:T}) \propto p(z_t, x_1:T) = p(z_t, x_{1:t}) p(x_{t+1:T}|z_t,x_{1:t}) = p(z_t, x_{1:t}) p(x_{t+1:T}|z_t) \propto \alpha_t(i) \beta_t(i)γt​=p(zt​∣x1:T​)∝p(zt​,x1​:T)=p(zt​,x1:t​)p(xt+1:T​∣zt​,x1:t​)=p(zt​,x1:t​)p(xt+1:T​∣zt​)∝αt​(i)βt​(i) since (xt+1:T⊥x1:t∣ztx_{t+1:T} \perp x_{1:t}|z_txt+1:T​⊥x1:t​∣zt​).

Forward Recursion​

We firstly define αt(j)=p(zt=j∣x1:t),t∈[1,T]\alpha_t(j) = p(z_t = j | x_{1:t}), t\in [1,T]αt​(j)=p(zt​=j∣x1:t​),t∈[1,T] and assume that p(z1),p(zt∣zt−1)p(z_1), p(z_t|z_{t-1})p(z1​),p(zt​∣zt−1​) and p(xt∣zt)p(x_t|z_t)p(xt​∣zt​) are known. Then we can compute αt(j)\alpha_t(j)αt​(j) recursively as follows:

  1. (Prediction step) compute the one-step-ahead predictive density (i.e. the new prior for time ttt)

    p(zt=j∣x1:t−1)=∑ip(zt=j∣zt−1=i)p(zt−1=i∣x1:t−1)=∑iΨ(i,j)αt−1(i)p(z_t = j | x_{1:t-1}) = \sum_i p(z_t = j | z_{t-1} = i) p(z_{t-1} = i | x_{1:t-1}) = \sum_i \Psi(i,j) \alpha_{t-1}(i)p(zt​=j∣x1:t−1​)=∑i​p(zt​=j∣zt−1​=i)p(zt−1​=i∣x1:t−1​)=∑i​Ψ(i,j)αt−1​(i).

  2. (Update step) αt(j)=p(zt=j∣x1:t)=p(zt=j∣x1:t−1,xt)∝p(xt∣zt=j,x1:t−1)p(zt=j∣x1:t−1)∝p(xt∣zt=j)p(zt=j∣x1:t−1)=ψt(j)p(zt=j∣x1:t−1)\alpha_t(j) = p(z_t = j | x_{1:t}) = p(z_t = j | x_{1:t-1}, x_t) \\\propto p(x_t |z_t=j, x_{1:t-1}) p(z_t = j | x_{1:t-1}) \\\propto p(x_t |z_t=j) p(z_t = j | x_{1:t-1}) \\ = \psi_t(j) p(z_t = j | x_{1:t-1})αt​(j)=p(zt​=j∣x1:t​)=p(zt​=j∣x1:t−1​,xt​)∝p(xt​∣zt​=j,x1:t−1​)p(zt​=j∣x1:t−1​)∝p(xt​∣zt​=j)p(zt​=j∣x1:t−1​)=ψt​(j)p(zt​=j∣x1:t−1​) with normalization constant Zt=p(xt∣x1:t−1)=∑jp(zt=j∣x1:t−1)p(xt∣zt=j)Z_t = p(x_t | x_{1:t-1}) = \sum_j p(z_t = j | x_{1:t-1}) p(x_t | z_t = j)Zt​=p(xt​∣x1:t−1​)=∑j​p(zt​=j∣x1:t−1​)p(xt​∣zt​=j).

Such algorith sometimes also called predict-update cycle. We can also use matrix notation where αt∝ψt⊙(ΨTαt−1)\alpha_t \propto \psi_t \odot (\Psi^T \alpha_{t-1})αt​∝ψt​⊙(ΨTαt−1​) where:

  • ψt=p(xt∣zt=j)\psi_t = p(x_t|z_t = j)ψt​=p(xt​∣zt​=j) is the local evidence at time ttt.
  • Ψ=p(zt+1=j∣zt=i)\Psi = p(z_{t+1} = j|z_t = i)Ψ=p(zt+1​=j∣zt​=i) is the transition matrix.
  • ⊙\odot⊙ is the element-wise product.

Backward Recursion​

We define βt(j)=p(xt+1:T∣zt=j)\beta_t(j) = p(x_{t+1:T}|z_t = j)βt​(j)=p(xt+1:T​∣zt​=j), then we have:

=∑jp(zt+1=j,xt+1:T∣zt=i)=∑jp(xt+2:T∣zt+1=j,zt=i,xt+1)p(xt+1∣zt+1=j,zt=i)p(zt+1=j∣zt=i)=∑jp(xt+2:T∣zt+1=j)p(xt+1∣zt+1=j)p(zt+1=j∣zt=i)=∑jβt+1(j)ψt+1(j)Ψ(i,j)= \sum_j p(z_{t+1} = j, x_{t+1:T}|z_t = i) \\ = \sum_j p(x_{t+2:T}|z_{t+1} = j, z_t = i,x_{t+1})p(x_{t+1}|z_{t+1} = j, z_t = i)p(z_{t+1} = j|z_t = i) \\ = \sum_j p(x_{t+2:T}|z_{t+1} = j)p(x_{t+1}|z_{t+1} = j)p(z_{t+1} = j|z_t = i) \\ = \sum_j \beta_{t+1}(j) \psi_{t+1} (j) \Psi(i,j)=∑j​p(zt+1​=j,xt+1:T​∣zt​=i)=∑j​p(xt+2:T​∣zt+1​=j,zt​=i,xt+1​)p(xt+1​∣zt+1​=j,zt​=i)p(zt+1​=j∣zt​=i)=∑j​p(xt+2:T​∣zt+1​=j)p(xt+1​∣zt+1​=j)p(zt+1​=j∣zt​=i)=∑j​βt+1​(j)ψt+1​(j)Ψ(i,j)

In matrix notation, βt=Ψ(ψt+1⊙βt+1)\beta_t = \Psi(\psi_{t+1} \odot \beta_{t+1})βt​=Ψ(ψt+1​⊙βt+1​) where βT(i)=1\beta_T(i) = 1βT​(i)=1.

Viterbi Algorithm​

In order to infer the most likely sequence of hidden states. (i.e. arg max⁡z1:Tp(z1:T∣x1:T)\argmax_{z_{1:T}} p(z_{1:T}|x_{1:T})argmaxz1:T​​p(z1:T​∣x1:T​)). We can use the Viterbi Algorithm.

In Viterbi algorithm, the forward pass does use max-product, but the backwards pass uses a traceback procedure to recover the most probable path.

Firstly, we define δt(i)=max⁡z1:t−1p(z1:t−1,zt=i,x1:t)\delta_t(i) = \max_{z_{1:t-1}} p(z_{1:t-1}, z_t = i, x_{1:t})δt​(i)=maxz1:t−1​​p(z1:t−1​,zt​=i,x1:t​) which is the probability of ending up in state jjj at time ttt, by taking the most probable path.

That is, δt(j)=max⁡z1:t−1p(z1:t−1,zt=j∣x1:t)∝max⁡z1:t−1p(z1:t−2,zt−1=i∣x1:t−1)p(zt=j∣zt−1=i)p(xt∣zt=j)=max⁡iδt−1(i)Ψ(i,j)ψt(j)\delta_t(j) = \max_{z_{1:t-1}} p(z_{1:t-1}, z_t = j| x_{1:t}) \propto \max_{z_{1:t-1}} p(z_{1:t-2}, z_{t-1} = i| x_{1:t-1}) p(z_{t}=j|z_{t-1}=i) p(x_t|z_t=j) = \max_i \delta_{t-1}(i) \Psi(i,j) \psi_t(j)δt​(j)=maxz1:t−1​​p(z1:t−1​,zt​=j∣x1:t​)∝maxz1:t−1​​p(z1:t−2​,zt−1​=i∣x1:t−1​)p(zt​=j∣zt−1​=i)p(xt​∣zt​=j)=maxi​δt−1​(i)Ψ(i,j)ψt​(j).

The most likely previous state is θt(j)=arg max⁡iδt−1(i)Ψ(i,j)\theta_t(j) = \argmax_i \delta_{t-1}(i) \Psi(i,j)θt​(j)=argmaxi​δt−1​(i)Ψ(i,j).

Then we have the following recursion algorithm:

  • We initial with δ1(i)=π(i)ψ1(i)\delta_1(i) = \pi(i) \psi_1(i)δ1​(i)=π(i)ψ1​(i) where πj=p(z1=j)\pi_j = p(z_1 = j)πj​=p(z1​=j) and terminate with zT∗=arg max⁡iδT(i)z_T^* = \argmax_i \delta_T(i)zT∗​=argmaxi​δT​(i).
  • Then we can compute the most probable sequence of states using traceback zt∗=θt+1(zt+1∗)z_t^* = \theta_{t+1}(z_{t+1}^*)zt∗​=θt+1​(zt+1∗​)
