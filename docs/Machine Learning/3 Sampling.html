Sampling

We have multiple ways to do sampling.

Ancestral Sampling​

The way sampling data based a joint distribution in a topological order is called ancestral sampling.

  • at each step, sample from any conditional distribution that you haven't visited yet whose parent have all been sampled

Rejection Sampling​

We have the other sampling method without using pdf p(x)p(x)p(x). Similar as importance sampling, we would like to use a simpler propsal density q(x)q(x)q(x) (can be evaluated within a multiplicative factor ZqZ_qZq​). We assume cq~(x)>p~(x)c\tilde{q}(x) > \tilde{p}(x)cq~​(x)>p~​(x) for all xxx. That is we have rejection sampling procedure follows:

  1. Generate two random variables
    1. Denote the one generated from the proposal density q(x)q(x)q(x) as xxx.
    2. Denote the one uniformly generated from the interval [0,cq~(x)][0, c\tilde{q}(x)][0,cq~​(x)] as uuu.
  2. Evaluate p~(x)\tilde{p}(x)p~​(x) and accept or reject the sample xxx by comparing the value of uuu with the value of p~(x)\tilde{p}(x)p~​(x).
    1. If u>p~(x)u > \tilde{p}(x)u>p~​(x), reject the sample xxx.
    2. else, accept the sample xxx. Add xxx into set of samples {x(r)}r=1R\{x^{(r)}\}^R_{r = 1}{x(r)}r=1R​ and discard uuu.

some facts about rejection sampling:

  • high-dimensional problems always lead huge ccc so that rare to accept samples
  • find ccc is hard
  • define acceptance rate =area under p~(x)/area under cq~(x)=1/Z= \text{area under } \tilde{p}(x) / \text{area under } c\tilde{q}(x) = 1/Z=area under p~​(x)/area under cq~​(x)=1/Z
  • the accepted point {x(r)}r=1R\{x^{(r)}\}^R_{r = 1}{x(r)}r=1R​ are independent.

Metropolis-Hastings Algorithm​

As we described above, importance and rejection sampling work only when if the proposal density q(x)q(x)q(x) is similar to the target density p(x)p(x)p(x) and we hard to find such q(X)q(X)q(X) in high-dimensional problems. We can use Metropolis-Hastings Algorithm to solve this problem where we make such q(x)q(x)q(x) depends on the current state xxx (i.e. q(x∣x(t))q(x|x^{(t)})q(x∣x(t))). The algorithm is as follows:

  • A tentative new state x′x'x′ is generated from the proposal density q(x′∣x(t))q(x'|x^{(t)})q(x′∣x(t)). We accept the new state with probability A(x′∣x(t))=min⁡{1,p~(x′)q(x(t)∣x′)p~(x(t))q(x′∣x(t))}A(x'|x^{(t)}) = \min\{1, \frac{\tilde{p}(x')q(x^{(t)}|x')}{\tilde{p}(x^{(t)})q(x'|x^{(t)})}\}A(x′∣x(t))=min{1,p~​(x(t))q(x′∣x(t))p~​(x′)q(x(t)∣x′)​}.

The facts about this algorithm:

  • q(x∣x(t))q(x|x^{(t)})q(x∣x(t)) is not necessarily looks like p(x)p(x)p(x)
  • This algorithm generalizes a dependent sequence
  • Each sample x(t)x^{(t)}x(t) has a pdf depends on the previous sample x(t−1)x^{(t-1)}x(t−1). (i.e. the markov chain)
  • need to be run for a time in order to generate samples from the target distribution p(x)p(x)p(x), i.e. do Monte Carlo estimation when TTT is large enough to estimate the mean of test function ϕ\phiϕ: Ex∼p(x)[ϕ(x)]=1T∑t=1Tϕ(x(t))\mathbb{E}_{x\sim p(x)}[\phi(x)] = \frac{1}{T}\sum_{t = 1}^T \phi(x^{(t)})Ex∼p(x)​[ϕ(x)]=T1​∑t=1T​ϕ(x(t))
  • The procedure defines a Markov chain with stationary distribution π(x)\pi(x)π(x) equal to the target distribution p(x)p(x)p(x) (i.e. π(x)=p(x)\pi(x) = p(x)π(x)=p(x))

      • sample xj(t)x_j^{(t)}xj(t)​ from the conditional distribution given other component xj(t)∼p(xj∣x−j(t))x_j^{(t)} \sim p(x_j|x_{-j}^{(t)})xj(t)​∼p(xj​∣x−j(t)​) where x−j(t)x_{-j}^{(t)}x−j(t)​ is the vector of all other components of x(t)x^{(t)}x(t) except xj(t)x_j^{(t)}xj(t)​.

Hamiltonian Monte Carlo​

Hamiltonian Monte Carlo (HMC) is a variant of the Metropolis-Hastings algorithm (with a specialized proposal mechanism). This algorithm uses a physical analogy to make proposals (the potential energy). Before we talk about HMC, we should know what is Hamiltonian.

  • Momentum ν\nuν carrying the kinetic energy K(ν)=12νTνK(\nu) = \frac{1}{2}\nu^T\nuK(ν)=21​νTν and it follows a Gaussian distribution and independent of position xxx.
  • Totaly enery or Hamiltonian H(x,ν)=E(x)+K(ν)H(x, \nu) = E(x) + K(\nu)H(x,ν)=E(x)+K(ν).
  • Energy is preserved:
    • Frictionless ball rolling: (x,ν)→(x′,ν′)(x, \nu) \to (x', \nu')(x,ν)→(x′,ν′)
    • H(x,ν)=H(x′,ν′)H(x, \nu) = H(x', \nu')H(x,ν)=H(x′,ν′)
  • Ideal Hamiltonian dynamics are reversible: reverse ν\nuν and the ball will return to its start point! (x,−ν)→(x′,−ν′)(x, -\nu) \to (x', -\nu')(x,−ν)→(x′,−ν′)
    • use dxdt=∂H∂ν=∂K∂ν\frac{dx}{dt} = \frac{\partial H}{\partial \nu} = \frac{\partial K}{\partial \nu}dtdx​=∂ν∂H​=∂ν∂K​ and dνdt=−∂H∂x=−∂E∂x\frac{d\nu}{dt} = -\frac{\partial H}{\partial x} = -\frac{\partial E}{\partial x}dtdν​=−∂x∂H​=−∂x∂E​.
    • dHdt=∑i∂E∂xidxidt+∑i∂K∂νidνidt=0\frac{dH}{dt} = \sum_i \frac{\partial E}{\partial x_i}\frac{d x_i}{d t} + \sum_i \frac{\partial K}{\partial \nu_i}\frac{d \nu_i}{d t} = 0dtdH​=∑i​∂xi​∂E​dtdxi​​+∑i​∂νi​∂K​dtdνi​​=0.
    • we can apply Leap-frog Integrator where is a numerical approximation of the Hamiltonian dynamics:
      • ν(t+0.5ϵ)=ν(t)+ϵ2d≠dt(t)=ν(t)−ϵ2∂E∂x(x(t))\nu(t + 0.5\epsilon) = \nu(t) + \frac{\epsilon}{2}\frac{d\ne}{dt}(t) = \nu(t) - \frac{\epsilon}{2}\frac{\partial E}{\partial x}(x(t))ν(t+0.5ϵ)=ν(t)+2ϵ​dtd=​(t)=ν(t)−2ϵ​∂x∂E​(x(t))
      • x(t+ϵ)=x(t)+ϵdxdt(t)=x(t)+ϵ∂K∂ν(ν(t+0.5ϵ))x(t + \epsilon) = x(t) + \epsilon\frac{dx}{dt}(t) = x(t) + \epsilon\frac{\partial K}{\partial \nu}(\nu(t + 0.5\epsilon))x(t+ϵ)=x(t)+ϵdtdx​(t)=x(t)+ϵ∂ν∂K​(ν(t+0.5ϵ))
      • ν(t+ϵ)=ν(t+0.5ϵ)−ϵ2∂E∂x(x(t+ϵ))\nu(t + \epsilon) = \nu(t + 0.5\epsilon) - \frac{\epsilon}{2}\frac{\partial E}{\partial x}(x(t + \epsilon))ν(t+ϵ)=ν(t+0.5ϵ)−2ϵ​∂x∂E​(x(t+ϵ))
  • The joint distribution p(x,ν)∝exp⁡(−H(x,ν))p(x, \nu) \propto \exp(-H(x, \nu))p(x,ν)∝exp(−H(x,ν)).

That is, the MCMC procedure is (run until mixing):

  1. start at (x(0),ν(0))(x^{(0)}, \nu^{(0)})(x(0),ν(0)).
  2. Sampel the momentum from the standard Gaussian distribution ν(t)∼N(0,I)\nu^{(t)} \sim \mathcal{N}(0, I)ν(t)∼N(0,I).
  3. continue at (x,ν)=(x(t−1),ν(t))(x, \nu) = (x^{(t-1)}, \nu^{(t)})(x,ν)=(x(t−1),ν(t)) for a short time interval ϵ\epsilonϵ and run the leap-frog integrator for LLL steps to reach (x′,ν′)(x', \nu')(x′,ν′)
  4. Accept the new state (x′,−ν′)(x', -\nu')(x′,−ν′) with probability min⁡{1,exp⁡(H(x(t−1),ν(t−1))/exp⁡(H(x′,ν′)))}\min\{1, \exp(H(x^{(t-1)}, \nu^{(t-1)})/\exp(H(x', \nu')))\}min{1,exp(H(x(t−1),ν(t−1))/exp(H(x′,ν′)))}. (lower energy is more likely to be accepted)
