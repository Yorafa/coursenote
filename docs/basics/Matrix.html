Matrix

Matrix is the extension of vector where from n×1→n×mn\times 1 \to n\times mn×1→n×m, but notice, the addition of matrix only perform when they are all the same n×mn\times mn×m where they have the same number of rows and columns. For multiplication, we can only perform An×p=Bn×k⋅Ck×pA_{n\times p} = B_{n\times k} \cdot C_{k\times p}An×p​=Bn×k​⋅Ck×p​

Properties​

Distributive property: A⋅(B+C)=A⋅B+A⋅CA \cdot (B + C) = A \cdot B + A \cdot CA⋅(B+C)=A⋅B+A⋅C and (A+B)⋅C=A⋅C+B⋅C(A + B) \cdot C = A \cdot C + B \cdot C(A+B)⋅C=A⋅C+B⋅C

Associative property: (A⋅B)⋅C=A⋅(B⋅C)(A \cdot B) \cdot C = A \cdot (B \cdot C)(A⋅B)⋅C=A⋅(B⋅C)

Identity property: A⋅I=AA \cdot I = AA⋅I=A and I⋅A=AI \cdot A = AI⋅A=A

Inverse property: A⋅A−1=IA \cdot A^{-1} = IA⋅A−1=I and A−1⋅A=IA^{-1} \cdot A = IA−1⋅A=I

The inverse of product of matrix: (AB)−1=B−1A−1(AB)^{-1} = B^{-1}A^{-1}(AB)−1=B−1A−1, but only square matrix has inverse we will see below

The transpose of product of matrix: (AB)T=BTAT(AB)^T = B^TA^T(AB)T=BTAT

  • (A−1)′=(A′)−1(A^{-1})' = (A')^{-1}(A−1)′=(A′)−1

We define the rank of a matrix AAA as the maximum number of linearly independent row/columns of AAA

Idempotent Matrices​

A square matrix is Idempotent Matrices if A2=AA^2 = AA2=A

  • I−AI - AI−A is also Idempotent
  • The trace of an idempotent matrix is always integer-valued.

Projection Matrices​

A square matrix AAA is an Orthogonal Projection Matrices if A2=A=A′A^2 = A = A'A2=A=A′

  • AAA also is Idempotent and symmetric
  • THe eigenvalues of a projection matrix are either 0 or 1

Diagonalizable Matrices​

A diagonal matrix is the matrix that has only non-zero elements on the main diagonal.

A square matrix AAA is diagonalizable iff there exists a invertible matrix PPP such that A=PDP−1A = PDP^{-1}A=PDP−1 where DDD is a diagonal matrix.

Normal Matrices​

A square matrix AAA is normal iff A∗A=AA∗A^*A = AA^*A∗A=AA∗ where A∗A^*A∗ is the conjugate transpose of AAA. That is, we only talk about normmal matrices over C\mathbb{C}C.

Decomposition​

Depend on different matrix, we have different decomposition. Every matrix has a Singular Value Decomposition (SVD). Every square matrix has a Jordan Decomposition. Every diagonalizable matrix has a eigenvalue decomposition when such factorized matrix is a normal or real symmetric matrix we call it spectral decomposition.
