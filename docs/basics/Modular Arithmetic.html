Modular Arithmetic

let m be a fixed natural number greater than 1. we say b modulo m if a - b is divisible by m. a≡b(modm)a\equiv b\pmod ma≡b(modm)

  1. a≡b(modm)a\equiv b\pmod ma≡b(modm), then b≡a (modm)b\equiv a\ \pmod mb≡a (modm)
  2. a≡b(modm)a\equiv b\pmod ma≡b(modm), and b≡c (modm)b\equiv c\ \pmod mb≡c (modm), then a≡c(modm)a\equiv c \pmod ma≡c(modm)
  3. a≡b(modm),a,b≥0  ⟺  a=k1m+r1,b=k2m+r2,r1=r2a\equiv b\pmod m, a,b\ge0 \iff a = k_1m+r_1, b = k_2m+r_2, r_1 =r_2a≡b(modm),a,b≥0⟺a=k1​m+r1​,b=k2​m+r2​,r1​=r2​
  4. ∀m,a≡b(modm),b∈{n∣0≤n<m}\forall m, a\equiv b \pmod m, b\in\{n| 0\le n < m\}∀m,a≡b(modm),b∈{n∣0≤n<m}]
  5. a≡b(modm),c≡d(modm)a\equiv b\pmod m, c\equiv d\pmod ma≡b(modm),c≡d(modm)
    1. (a+c)≡(b+d)(modm)(a + c) \equiv (b + d) \pmod m(a+c)≡(b+d)(modm)
    2. ac≡bd(modm)ac \equiv bd \pmod mac≡bd(modm)
  6. a≡b(modm)  ⟹  an≡bn(modm),n∈Na\equiv b\pmod m\implies an\equiv bn\pmod m, n\in \mathbb{N}a≡b(modm)⟹an≡bn(modm),n∈N
