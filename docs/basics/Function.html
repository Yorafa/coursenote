Function

We say the way transfer an element of a collection to the other collection as function. In mathematic, we always use function to transfer an element of a set to the other set. (Obviously set is a kind of collection.) Or sample space in statistics, or parameters in CS.

Generally, given two sets AAA and BBB, we define a function from AAA to BBB is a rule or mapping that takes each element x∈Ax\in Ax∈A and associate with it a single element of BBB.

Let fff be function map AAA to BBB, or we also can say fff is a function from AAA to BBB where:

  • AAA is the domain of this function (Input)
  • BBB is the codomain of this function (output)
  • ∀a∈A,f(a)∈B\forall a\in A, f(a)\in B∀a∈A,f(a)∈B

Injective function or 1→11\to 11→1:

  • A function f:A→Bf: A\to Bf:A→B, ∀x,y∈A.(x≠y  ⟹  f(x)≠f(y))\forall x,y \in A. (x\ne y \implies f(x) \ne f(y))∀x,y∈A.(x=y⟹f(x)=f(y))
  • A function f:A→Bf: A\to Bf:A→B, ∀x,y∈A.(f(x)=f(y)  ⟹  x=y)\forall x,y \in A. (f(x) = f(y) \implies x = y)∀x,y∈A.(f(x)=f(y)⟹x=y)
  • The Pigeonhole Principle: Let A,BA, BA,B be sets, ∣A∣>∣B∣|A|>|B|∣A∣>∣B∣   ⟹  \implies⟹ no such injective function f:A→Bf:A\to Bf:A→B
    • In other language, we present this theory as: If there are mmm pigeon and nnn pigeonholes, not matter how we assign pigeons to pigeonholes, at least one pigeonhole will have at least ⌈m/n⌉\lceil m/n \rceil⌈m/n⌉

Surjective function or onto:

  • A function f:A→Bf: A\to Bf:A→B, ∀b∈B,∃a∈A(f(a)=b)\forall b \in B, \exists a\in A (f(a) = b)∀b∈B,∃a∈A(f(a)=b)
  • Let A,BA, BA,B be sets, ∣A∣<∣B∣|A|<|B| ∣A∣<∣B∣   ⟹  \implies⟹ no such surjective function f:A→Bf:A\to Bf:A→B (reverse pigeonhole principle)

hits times: a∈A,b∈B,ba\in A, b \in B, ba∈A,b∈B,b is hit k times by function fff if there are k distinct a∈Aa \in Aa∈A are such that f(a)=bf(a) = bf(a)=b or we can say ∣{a∈A:f(a)=b}∣=k|\{a\in A: f(a) = b\}| = k∣{a∈A:f(a)=b}∣=k

Continuous Functions on Compact Sets​

A subset S⊂RS\subset\mathbb{R}S⊂R may have a boundary or be bounded:

  • If it is bounded above, then ∃M∈R,∀s∈S,s≤M\exists M\in\mathbb{R},\forall s\in S, s\le M∃M∈R,∀s∈S,s≤M. MMM in this case is the upper bound for SSS.

    • If ∀L∈R,L\forall L\in \mathbb{R}, L∀L∈R,L is a upper bound of SSS, M≤LM\le LM≤L then we call MMM is the least upper bound (supremum)

  • If it is bounded below, then ∃M∈R,∀s∈S,s≥M\exists M\in\mathbb{R},\forall s\in S, s\ge M∃M∈R,∀s∈S,s≥M. MMM in this case is the lower bound for SSS.

    • If ∀G∈R,G\forall G\in \mathbb{R}, G∀G∈R,G is a upper bound of SSS, M≥GM\ge GM≥G then we call MMM is the greatest lower bound (infimum)

  • If a set is bounded above and below, then it's a bounded set.

  • If a set has a maximum, then the set is bounded above and the maximum is the least upper bound.

  • If a set has a minimum, then the set is bounded below and the minimum is the greatest lower bound.

Least Upper Bound Principle:

  • Every nonempty subset of R\mathbb{R}R is bounded above has a supremum.
  • Every nonempty subset of R\mathbb{R}R is bounded below has a infimum.
