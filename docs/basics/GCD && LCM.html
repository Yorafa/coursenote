GCD && LCM

GCD​

GCD(Great Common Divisor):

  • gcd(a,b)=d  ⟹  ∃n,m∈Z,an+bm=dgcd(a,b)=d \implies \exists n,m\in \Z, an + bm = dgcd(a,b)=d⟹∃n,m∈Z,an+bm=d
  • let n,m∈Z,min⁡(an+bm)=gcd(a,b)n,m \in \Z, \min(an+bm) = gcd(a,b)n,m∈Z,min(an+bm)=gcd(a,b)
  • We can also extent the definition to gcd a set of numbers

Co-prime:

  • gcd(a,b)=1  ⟹  a,bgcd(a,b) = 1 \implies a,bgcd(a,b)=1⟹a,b are co-prime

LCM(Least Common Multiple):
