Induction

Almost every courses teach induction

Basic Induction: To prove ∀n∈N,(P(n))\forall n\in \N, (P(n))∀n∈N,(P(n)) use some base case and induction to prove for all.

  • (P(0)∧∀k∈N.(P(k)  ⟹  P(k+1)))  ⟹  ∀n∈N,(P(n))(P(0) \land \forall k \in \N. (P(k)\implies P(k+1))) \implies \forall n\in \N, (P(n))(P(0)∧∀k∈N.(P(k)⟹P(k+1)))⟹∀n∈N,(P(n))
  • we have some flexibility on basic case where if we have goal to prove ∀n≥1\forall n\ge1∀n≥1 then we will prove P(1)P(1)P(1)

Basic Induction prove template:

Prove by induction.
Base case: [Prove P(0) is true]
Inductive step: Let k be arbitrary natural number, and assume P(k). we will show P(k+1)

Complete Induction: To prove ∀n∈N,(P(n))\forall n\in \N, (P(n))∀n∈N,(P(n)) use some base case and more induction steps to prove for all.

  • (P(0)∧∀k∈N.(P(0)∧P(1)∧…∧P(k)  ⟹  P(k+1)))  ⟹  ∀n∈N,(P(n))(P(0) \land \forall k \in \N. (P(0)\land P(1) \land \ldots\land P(k)\implies P(k+1))) \implies \forall n\in \N, (P(n))(P(0)∧∀k∈N.(P(0)∧P(1)∧…∧P(k)⟹P(k+1)))⟹∀n∈N,(P(n))
  • similarly, we have flexibility on basic case.

WOP(Well Ordering Principle): ∀S⊆N(S≠∅),S\forall S\subseteq \N(S\ne \emptyset), S∀S⊆N(S=∅),S has a minimal element.

  • let a∈S,∀b∈S,a≤b  ⟹  aa\in S, \forall b\in S, a\le b \implies aa∈S,∀b∈S,a≤b⟹a is a minimal element

We can use WOP to prove induction statements of the form ∀n∈N,(P(n))\forall n\in \N, (P(n))∀n∈N,(P(n)):

  1. Check P(b)P(b)P(b) is true, bbb stand the basic case
  2. By contradiction, assume ∃n∈N.(¬P(n))\exists n\in \N.(\lnot P(n))∃n∈N.(¬P(n)). So the set S={n∈N:¬P(n)}S = \{n\in\N: \lnot P(n)\}S={n∈N:¬P(n)} is non-empty
  3. By the WOP, SSS has a minimal element, denote as mmm; commonly, mmm is the smallest natural number for which PPP doesn't hold.
  4. Derive the contradiction by showing P(m)P(m)P(m) or by finding a m′<mm' < mm′<m, for which ¬P(m′)\lnot P(m')¬P(m′)
