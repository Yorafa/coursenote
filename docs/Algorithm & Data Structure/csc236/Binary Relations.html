Binary Relations

Binary Relations: Denote a relation over a set AAA be RRR, if the relation is binary relation, then ∀a,b∈A\forall a,b \in A∀a,b∈A

  • if aaa relates bbb under relation RRR, denote as aRbaRbaRb
  • else aR̸ba\not R baRb

Some Property of Binary Relations RRR over a set AAA may have:

  • irreflexive: ∀a∈A,aR̸a\forall a \in A, a\not R a∀a∈A,aRa
  • reflexive: ∀a∈A,aRa\forall a \in A, aRa∀a∈A,aRa
  • asymmetric: ∀a,b∈A,aRb  ⟹  bR̸a\forall a,b \in A, aRb\implies b\not R a∀a,b∈A,aRb⟹bRa
  • symmetric: ∀a,b∈A,aRb  ⟹  bRa\forall a,b \in A, aRb\implies bRa∀a,b∈A,aRb⟹bRa
  • antisymmetric: ∀a,b∈A(aRb∧bRa  ⟹  a=b)\forall a,b\in A (aRb\land bRa \implies a=b)∀a,b∈A(aRb∧bRa⟹a=b)
  • transitive: ∀a,b,c∈A,aRb∧bRc  ⟹  aRc\forall a,b,c \in A, aRb \land bRc \implies aRc∀a,b,c∈A,aRb∧bRc⟹aRc
  • connected: ∀a,b∈A,a≠b  ⟹  aRb∨bRa\forall a,b \in A, a\ne b \implies aRb \lor bRa∀a,b∈A,a=b⟹aRb∨bRa
  • cyclic: ∀a,b,c∈A,aRb∧bRc  ⟹  cRa\forall a,b,c\in A, aRb\land bRc \implies cRa∀a,b,c∈A,aRb∧bRc⟹cRa
  • irreflexive ∧\land∧ transitive   ⟹  \implies⟹ asymmetric

Strict (partial) Order: a binary relation RRR over a set AAA is irreflexive, asymmetric(not necessary) and transitive

  • e.g.: Dependencies, Little-o, ⊂\sub⊂, etc.

Strict Total Order: a binary relation RRR over a set AAA is strict order and connected

  • e.g.: Dictionary Order, etc.

Hasse Diagram: ∀R,R\forall R, R∀R,R is strict order over a set AAA, ∀a,b∈A\forall a,b \in A∀a,b∈A, Hasse diagram has a edge a→b  ⟺  aRb∧∄c,aRc∧cRba\to b \iff a Rb \land \nexists c, aRc \land cRba→b⟺aRb∧∄c,aRc∧cRb

Notice: it (binary relation) is very different with the binary operation of group theory.

Partial Order: a binary relation RRR over a set AAA is reflexive, antisymmetric and transitive

Total Order: a binary relation RRR over a set AAA is partial order and connected.

Equivalence Relation: a binary relation RRR over a set AAA is reflexive, symmetric and transitive

  • e.g. ===, modular (≡n):a=b(mod n),a,b,n∈N(\equiv_n): a =b(mod \ n), a,b,n\in \N(≡n​):a=b(mod n),a,b,n∈N
  • cyclic and reflexive R  ⟹  R \impliesR⟹ RRR is equivalence relation

Equivalence Class: Let RRR be Equivalence Relation on a set AAA, ∀a∈A\forall a\in A∀a∈A, the Equivalence Class of aaa respect to RRR, denoted [a]R[a]_R[a]R​ has property [a]R={b∈A:aRb}[a]_R = \{b\in A: aRb\}[a]R​={b∈A:aRb}

  • ∀a,b∈A,[a]R=[b]R  ⟺  aRb\forall a,b \in A, [a]_R= [b]_R \iff aRb∀a,b∈A,[a]R​=[b]R​⟺aRb (lemma)

Partition: let AAA be any set, and XXX be a collection of subsets of AAA. ∀a∈A,∃S∈X,a∈S  ⟹  X\forall a\in A, \exists S\in X, a\in S \implies X∀a∈A,∃S∈X,a∈S⟹X is a partition of AAA

  • Let RRR be an equivalence relation on a set AAA. The set of Equivalence Classes is a partition of AAA.
  • Or we can say ∀a∈A,∃[a]R,a∈[a]R,[a]R\forall a \in A, \exists [a]_R, a\in[a]_R, [a]_R∀a∈A,∃[a]R​,a∈[a]R​,[a]R​ is unique
