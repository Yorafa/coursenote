Graph

A graph G=(V,E)G= (V, E)G=(V,E) is a pair of sets (V,E)(V, E)(V,E) where VVV is a set of vertices and EEE is a set of edges.

  • If EEE is a set of unordered pairs, the graph is called undirected. If EEE is a set of ordered pairs, the graph is called directed.
    • self-loop does not allowed in undirected graph (in csc236/csc263)
  • Weight of edge is a function w:E→Rw : E \to \Rw:E→R where w({u,v})w(\{u,v\})w({u,v}) is the weight of edge {u,v}\{u,v\}{u,v}
  • A matching M⊆EM\subseteq EM⊆E is a subset of edges that those edges that do not share any end points
    • every vertex in one matching then this matching is perfect
  • two vertices u,vu,vu,v are adjacent if {u,v}∈E\{u,v\}\in E{u,v}∈E
    • one is called neighbor of the other
    • an edge (u,v) is incident on vertices u and v. In a directed graph, the terminology differentiates between the beginning and ending vertex of an edge. So edge (u,v) which leaves vertex u is said to be incident from vertex u and is incident to (or enters) vertex v
  • A sequence of distinct vertices (v1,…,vn)(v_1,\ldots, v_n)(v1​,…,vn​) is a path from v1v_1v1​ to vnv_nvn​ if for every i∈{1,…,n−1},vii\in \{1,\ldots, n-1\}, v_ii∈{1,…,n−1},vi​ and vi+1v_{i+1}vi+1​ are adjacent.
    • The length of the path is the number of edges in the path.
    • A simple path contains no repeated edge
  • A sequence of distinct vertices (v1,…,vn)(v_1,\ldots, v_n)(v1​,…,vn​) is a the cycle if (v1,…,vn)(v_1,\ldots, v_n)(v1​,…,vn​) is a path, v1=vnv_1 = v_nv1​=vn​ and {vn−1,vn∈E}\{v_{n-1}, v_n\in E\}{vn−1​,vn​∈E}
    • A cycle is called Hamiltonian if every vertex appear in the cycle exactly once (except for the start/end vertex which appears twice)
    • A simple circle contains no repeated edge
  • A kkk-(vertex) coloring of GGG is a function f:V→Cf: V\to Cf:V→C , ∀u,v∈V,{u,v}∈E∨{v,u}∈E  ⟹  f(u)≠f(v)\forall u,v \in V, \{u,v\}\in E \lor \{v,u\}\in E \implies f(u) \ne f(v)∀u,v∈V,{u,v}∈E∨{v,u}∈E⟹f(u)=f(v)
  • Connected: ∀u,v∈V,v≠u\forall u,v \in V, v\ne u∀u,v∈V,v=u there is some path from uuu to vvv
  • acyclic: no cycle in graph
  • independent set: I⊆VI \subseteq VI⊆V , ∀v,u∈I,∄e∈E,e={v,u}∨e={u,v}  ⟹  I\forall v,u\in I, \nexists e\in E, e = \{v,u\}\lor e=\{u,v\} \implies I∀v,u∈I,∄e∈E,e={v,u}∨e={u,v}⟹I is an independent set
  • In an undirected graph, the degree of a vertex ν\nuν is the number of edges incident on ν\nuν . In a directed graph, the in-degree of vertex ν\nuν is the number of edges incident to ν\nuν (the size of set {(x,ν):x∈E}\{(x,\nu):x\in E\}{(x,ν):x∈E}) and the out-degree is the number of edges incident from v (the size of set {(ν,x):x∈E}\{(\nu,x):x\in E\}{(ν,x):x∈E}).
  • A connected component is a group of nodes that are connected by edges

Tree: a graph are connected and acyclic

  • Trees   ⟺  \iff⟺ minimally connected graph aka remove any edge will cause disconnect
  • Trees   ⟺  \iff⟺ maximally acyclic graph aka add any edge will cause cycle
  • ∀u,v∈V,u≠v  ⟹  \forall u,v\in V, u\ne v \implies∀u,v∈V,u=v⟹ unique path from uuu to vvv
  • Free tree is used for an undirected connected acyclic graph without a specific vertex designated as the root.
  • Rooted tree with one vertex is designated as the root.

A forest is a collection of (0+) disjoint trees

  • A tree is also a forest.
  • Total vertices - Total Edges = # trees

Binary Tree: a tree that every vertex has at most two edges incident to it.

Some examples about Graph:

  • locations on maps, relationship between people(contact facing), Courses, WIFI Connection, Trees, vector graph, airport routes, functions, binary relations

Matching Problem:

  • input: a graph with weight
  • output: A matching (usually maximum/minimum)

Pathing Finding Problem:

  • Input: a graph and two vertices
  • output: A path between two vertices

Travelling Salesman Problem:

  • Input: a graph and a start vertex
  • Output: a Hamiltonian cycle minimizes the total edge weights

Coloring Problem:

  • input: a graph
  • output a k-coloring of the graph

Independent Set Problem:

  • input: a graph, a number k∈Nk\in \Nk∈N with k>1k>1k>1
  • output: An independent set I⊆V∈GI \subseteq V\in GI⊆V∈G of size kkk
