Markov Chain

Markov Chain theorems are concerned with what will happen in a long run.

A (discrete time, discrete space, time-homogeneous(transition probability will not change whatever time change, that is, not depend on nnn)) Markov Chain is specified by the following three ingredients:

  1. A state space SSS, any non-empty finite or countbale set
  2. initial probabilities {νi}i∈S\{\nu_i\}_{i\in S}{νi​}i∈S​, where νi\nu_iνi​ is the probabiity of starting at state iii at time 0. Thus νi≥0\nu_i \ge 0νi​≥0 and ∑iνi=1\sum_{i} \nu_i = 1∑i​νi​=1
  3. Transition Probabilities {pij}i,j∈S\{p_{ij}\}_{i,j\in S}{pij​}i,j∈S​, where pijp_{ij}pij​ is the probability of jumping to state iii to state jjj. Thus pij≥0p_{ij} \ge 0pij​≥0 and ∑jpij=1\sum_{j} p_{ij} = 1∑j​pij​=1 for all iii.

We can write markov chain a sequence of random variables X0,X1,X2,…X_0, X_1, X_2, \dotsX0​,X1​,X2​,… where X0X_0X0​ is the initial state and XnX_nXn​ is the state at time nnn.

  • Initial Probabilities: νi=P(X0=i)∀i∈S\nu_i = P(X_0 = i) \forall i \in Sνi​=P(X0​=i)∀i∈S
  • the transition probabilities pijp_{ij}pij​ are basically conditional probability (conditional on starting). If P(Xn=i>0)P(X_n= i > 0)P(Xn​=i>0), then P(Xn+1=j∣Xn=i)=pijP(X_{n+1} = j | X_n = i) = p_{ij}P(Xn+1​=j∣Xn​=i)=pij​. pijp_{ij}pij​ does not depend on nnn. (again the time-homogeneous property)
  • P(Xn+1=j∣Xn=in,Xn−1=in−1,…,X1=i1,X0=i0)=P(Xn+1=j∣Xn=in)=pinjP(X_{n+1} = j|X_n = i_n, X_{n-1} = i_{n-1}, \dots, X_1 = i_1, X_0 = i_0) = P(X_{n+1} = j|X_n = i_n) = p_{i_nj}P(Xn+1​=j∣Xn​=in​,Xn−1​=in−1​,…,X1​=i1​,X0​=i0​)=P(Xn+1​=j∣Xn​=in​)=pin​j​ for all n≥0n \ge 0n≥0 and j,in,in−1,…,i1,i0∈Sj, i_n, i_{n-1}, \dots, i_1, i_0 \in Sj,in​,in−1​,…,i1​,i0​∈S (Markov Property). The equation basically says that the probability of the next state depends only on the current state, not on the past states. Also states that pij=P(Xn+1=j∣Xn=i)p_{ij} = P(X_{n+1} = j|X_n = i)pij​=P(Xn+1​=j∣Xn​=i).
  • P(Xn=in,Xn−1=in−1,…,X1=i1,X0=i0)=νi0pi0i1pi1i2…pin−1inP(X_n = i_n, X_{n-1} = i_{n-1}, \dots, X_1 = i_1, X_0 = i_0) = \nu_{i_0} p_{i_0i_1} p_{i_1i_2} \dots p_{i_{n-1}i_n}P(Xn​=in​,Xn−1​=in−1​,…,X1​=i1​,X0​=i0​)=νi0​​pi0​i1​​pi1​i2​​…pin−1​in​​
  • Denote μi(n)=P(Xn=i)\mu^{(n)}_i = P(X_n = i)μi(n)​=P(Xn​=i), then μi(n)=∑jμj(n−1)pji\mu^{(n)}_i = \sum_{j} \mu^{(n-1)}_j p_{ji}μi(n)​=∑j​μj(n−1)​pji​ for all n≥1n \ge 1n≥1 and i∈Si \in Si∈S. μi(n)\mu^{(n)}_iμi(n)​ present the end of nth steps with state i. (e.g. P(X1=1)=μ1(1)=∑jνjpj1P(X_1 = 1) = \mu^{(1)}_1 = \sum_{j} \nu_j p_{j1}P(X1​=1)=μ1(1)​=∑j​νj​pj1​)
    • μ(n)=νPn=μ(0)Pn\mu^{(n)} = \nu P^n = \mu^{(0)} P^nμ(n)=νPn=μ(0)Pn
  • pij(n)=P(Xn=j∣X0=i)=P(Xm+n=j∣Xm=i)=Pi(Xn=j)p_{ij}^{(n)} = P(X_n = j|X_0 = i) = P(X_{m+n}=j|X_m = i) = P_i(X_n = j)pij(n)​=P(Xn​=j∣X0​=i)=P(Xm+n​=j∣Xm​=i)=Pi​(Xn​=j)

Markov Chain Property​

A state iii is called recurrent if Pi(Xn=i for infinitely many n)=1P_i(X_n = i \text{ for infinitely many }n) = 1Pi​(Xn​=i for infinitely many n)=1 (or equivalently ∑n=1∞pii(n)=∞\sum_{n = 1}^{\infty} p_{ii}^{(n)} = \infty∑n=1∞​pii(n)​=∞). A state iii is called transient if Pi(Xn=i for infinitely many n)=0P_i(X_n = i \text{ for infinitely many }n) = 0Pi​(Xn​=i for infinitely many n)=0 (or equivalently ∑n=1∞pii(n)<∞\sum_{n = 1}^{\infty} p_{ii}^{(n)} < \infty∑n=1∞​pii(n)​<∞). More easily, a state iii is recurrent if fii=1f_{ii} = 1fii​=1 for all n≥1n \ge 1n≥1, otherwise transient.

Let N(i)N(i)N(i) be a random variable that present the total number of times the chain hits iii (not counting time 0).

  • let fijf_{ij}fij​ present return probability from state iii to jjj, then fij=Pi(N(j)≥1)f_{ij} = P_i(N(j) \ge 1)fij​=Pi​(N(j)≥1) and fijfjj(n−1)=Pi(N(j)≥n)f_{ij}f_{jj}^{(n - 1)} = P_i(N(j) \ge n)fij​fjj(n−1)​=Pi​(N(j)≥n)
  • The complementary probability 1−fij=Pi(Xn≠j∀n≥1)1 - f_{ij} = P_i(X_n \ne j \forall n \ge 1)1−fij​=Pi​(Xn​=j∀n≥1)
  • Pi(chain will eventually visit j and then eventually visit k)=fijfjkP_i(\text{chain will eventually visit }j \text{ and then eventually visit } k) = f_{ij}f_{jk}Pi​(chain will eventually visit j and then eventually visit k)=fij​fjk​

f - expansion: fij=pij+∑k≠jpikfkjf_{ij} = p_{ij} + \sum_{k\ne j} p_{ik}f_{kj}fij​=pij​+∑k=j​pik​fkj​

Communicating States:

  • State iii communicates with state jjj, written i→ji\to ji→j , if fij>0f_{ij} > 0fij​>0, i.e., if it is possible to get from iii to jjj.
  • Let fij>0  ⟺  ∃m≥1f_{ij} > 0 \iff \exists m\ge 1fij​>0⟺∃m≥1 with pij(m)>0p_{ij}^{(m)} > 0pij(m)​>0, i.e., there is some time mmm for which it is possible to get from iii to jjj in mmm steps.
  • We will write i↔ji \leftrightarrow ji↔j if i→ji \to ji→j and j→ij \to ij→i.

A Markov Chain is irreducible if i→j,∀i,j∈Si \to j, \forall i,j \in Si→j,∀i,j∈S, i.e., if fij>0,∀i,j∈Sf_{ij} > 0, \forall i,j \in Sfij​>0,∀i,j∈S. Otherwise, it is reducible.

If i→ki\to ki→k and l→jl\to jl→j, and ∑n=1∞pkl(n)=∞\sum_{n=1}^{\infty} p_{kl}^{(n)} = \infty∑n=1∞​pkl(n)​=∞, then ∑n=1∞pij(n)=∞\sum_{n=1}^{\infty} p_{ij}^{(n)} = \infty∑n=1∞​pij(n)​=∞.

If i↔ki\leftrightarrow ki↔k, then iii is recurrent iff kkk is recurrent.

Stationary Distribution: If π\piπ is a probability distribution on SSS, i.e., ∀i∈S,πi≥0\forall i \in S, \pi_i \ge 0∀i∈S,πi​≥0 ans ∑i∈Sπi=1\sum_{i \in S} \pi_i = 1∑i∈S​πi​=1, then π\piπ is a stationary distribution for a Markov Chain with transition probabilities pijp_{ij}pij​ if πi=∑j∈Sπjpji\pi_i = \sum_{j \in S} \pi_j p_{ji}πi​=∑j∈S​πj​pji​. In matrix form, π=πP\pi = \pi Pπ=πP.

  • A markov chain with stationary distribution π\piπ. If such chain start with probabilities πi\pi_iπi​, then it will keep the same probabilities one time uint later
  • π\piπ also is a eigenvector with eigenvalue λ=1\lambda = 1λ=1
  • if μ(0)=ν=π\mu^{(0)} = \nu = \piμ(0)=ν=π, then μ(1)=π\mu^{(1)} = \piμ(1)=π and μ(n)=π\mu^{(n)} = \piμ(n)=π for all n≥1n \ge 1n≥1.

∣S∣<∞,∀i,j∈S,∑i∈Spij=1,∑j∈Spij=1  ⟹  |S| < \infty, \forall i,j \in S, \sum_{i\in S}p_{ij} = 1, \sum_{j\in S}p_{ij} = 1 \implies∣S∣<∞,∀i,j∈S,∑i∈S​pij​=1,∑j∈S​pij​=1⟹ such markov chain is doubly stochastic.

A Markov Chain is reversible or time reversible with respect to probability distribution π\piπ, if ∀i,j∈S,πipij=πjpji\forall i,j \in S, \pi_ip_{ij} = \pi_jp_{ji}∀i,j∈S,πi​pij​=πj​pji​.

  • use to understand whether stationary distribution exists.
  • A chain is reversible with respect to π  ⟹  π\pi\implies \piπ⟹π is a stationary distribution. (converse is not true)
    • e.g. p12=p21=p31=1p_{12} = p_{21} = p_{31} = 1p12​=p21​=p31​=1, π=(1/3,1/3,1/3)\pi = (1/3, 1/3, 1/3)π=(1/3,1/3,1/3). Then we have π1p13≠π3p31\pi_{1} p_{13} \ne \pi_{3} p_{31}π1​p13​=π3​p31​, so it is not reversible.

Markov Chain with transitions pijp_{ij}pij​ on a state space SSS, and a state i∈Si\in Si∈S, the period of iii is the greatest common divisor of the set {i:∀n≥1,pii(n)=P(Xn=i∣X0=i)}\{i: \forall n\ge 1, p_{ii}^{(n)} = P(X_n = i| X_0 = i)\}{i:∀n≥1,pii(n)​=P(Xn​=i∣X0​=i)}.

  • A chain is aperiodic if the period of every state is 1.
