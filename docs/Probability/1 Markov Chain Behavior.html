Markov Chain Behavior

As we learn more about Markov chains (especially the infinite ones), we would like to find out the behavior of the chain. One of the interesting behavior is the stationary distribution.

Stationary Distribution​

Sometimes we find there exists π‾\overline{\pi}π satisfied π‾P=π‾\overline{\pi} P = \overline{\pi}πP=π where PPP is the transition matrix. We define such π‾\overline{\pi}π is called stationary distribution. From the equation, we not hard to conclude that once MC reach the distribution, it will never leave.

  • Assume π‾P=π‾\overline{\pi} P = \overline{\pi}πP=π, then we have π‾P=π‾PP=π‾P⋯P=π‾P(n)=π‾\overline{\pi} P = \overline{\pi} P P = \overline{\pi} P \cdots P = \overline{\pi} P^{(n)} = \overline{\pi}πP=πPP=πP⋯P=πP(n)=π.

We define a MC with probability distribution π‾\overline{\pi}π satisfies πipij=πjpji,∀i,j∈S\pi_i p_{ij} = \pi_j p_{ji}, \forall i,j \in Sπi​pij​=πj​pji​,∀i,j∈S is called reversible.

  • If a MC is reversible, then such π‾\overline{\pi}π is the stationary distribution. (converse is not true)

But we actually may not have such stationary distribution when following conditions are satisfied:

  • ∀i,j∈S,lim⁡n→∞pij(n)=0\forall i,j \in S,\lim_{n \to \infty} p_{ij}^{(n)} = 0∀i,j∈S,limn→∞​pij(n)​=0
  • MC is irreducible with some i,j∈Si,j\in Si,j∈S such that lim⁡n→∞pij(n)=0\lim_{n \to \infty} p_{ij}^{(n)} = 0limn→∞​pij(n)​=0

Finite MC convergence to stationary distribution​

Since PPP is a transition matrix, we would like to use some linear algebra ways to find the limit behavior of the MC, i.e. lim⁡n→∞P(n)\lim_{n \to \infty} P^{(n)}limn→∞​P(n). We would like to use Jordan decomposition to represent PPP where ∃Q\exists Q∃Q such that P=QDQ−1P = QDQ^{-1}P=QDQ−1 where D=Q−1PQD = Q^{-1}PQD=Q−1PQ. Then we have lim⁡n→∞P(n)=lim⁡n→∞QDnQ−1\lim_{n \to \infty} P^{(n)} = \lim_{n \to \infty} QD^nQ^{-1}limn→∞​P(n)=limn→∞​QDnQ−1

And we can use Perron-Frobenius theorem to find the stationary distribution since PPP is a stochastic matrix with all of entries are strictly positive, that is, all eigenvalues of PPP has absolute value less than 1. The columns of QQQ are right eigenvectors of PPP and the rows of Q−1Q^{-1}Q−1 are left eigenvectors of PPP.

Theorem 1​

According to above, we have the fact/theorem: If PPP is a transition matrix such that for some n,P(n)n, P^{(n)}n,P(n) has all entries strictly positive, then PPP statisfies:

  • The left eigenvector can be chosen to have all nonnegative entries.
  • The eigenvalue 1 is simple and all other eigenvalues have absolute value less than 1.
  • The first row of Q−1Q^{-1}Q−1 is the stationary distribution.

Theorem 2​

To find all entries strictly positive P(n)P^{(n)}P(n), we can use the following theorem: If PPP is irreducible and aperiodic, then ∃M>0\exists M > 0∃M>0 such that for all n≥M,P(n)n \geq M, P^{(n)}n≥M,P(n) has all entries strictly positive.

But what if PPP is not irreducible and aperiodic? We divide into following cases:

PPP is not irreducible or aperiodic​

Since PPP is reducible with recurrent classes R1,…,RrR_1, \ldots, R_rR1​,…,Rr​ transient classes T1,…,TsT_1, \ldots, T_sT1​,…,Ts​. Then we will have rrr stationary distributions π1,…,πr\pi_1, \ldots, \pi_rπ1​,…,πr​ corresponding to recurrent classes R1,…,RrR_1, \ldots, R_rR1​,…,Rr​. Let PkP_kPk​ be submatrix of PPP corresponding to recurrent classes RkR_kRk​. Then we have ∀i∈Rk,lim⁡n→∞Pij(n)=πk(j)∀j∈Rk\forall i \in R_k, \lim_{n \to \infty} P^{(n)}_{ij} = \pi_k(j) \forall j \in R_k∀i∈Rk​,limn→∞​Pij(n)​=πk​(j)∀j∈Rk​ and lim⁡n→∞Pij(n)=0∀j∉Rk\lim_{n \to \infty} P^{(n)}_{ij} = 0 \forall j \notin R_klimn→∞​Pij(n)​=0∀j∈/Rk​.

if state iii is transient, then lim⁡n→∞Pij(n)=0\lim_{n \to \infty} P^{(n)}_{ij} = 0limn→∞​Pij(n)​=0 for any other transient state jjj, and lim⁡n→∞Pij(n)=αk(i)πk(j)\lim_{n \to \infty} P^{(n)}_{ij} = \alpha_k(i) \pi_k(j)limn→∞​Pij(n)​=αk​(i)πk​(j) for any recurrent state jjj.

That is, we have lim⁡n→∞νPij(n)=αk(i)πk(j)\lim_{n \to \infty} \nu P^{(n)}_{ij} = \alpha_k(i) \pi_k(j)limn→∞​νPij(n)​=αk​(i)πk​(j) exists, where ν\nuν is initial probability distribution.

PPP is irreducible but periodic​

Since the chain periodic, we can split the state into ddd classes A1,…,AdA_1, \ldots, A_dA1​,…,Ad​ such that each state class has the same period. Then PPP will have ddd eigenvalue of absolute value 1, ddd complex number zzz with zd=1z^d = 1zd=1. That is, given initial probability distribution ν\nuν, we have the stationary distribution lim⁡n→∞1d∑k=1dνP(n+k)=π\lim_{n \to \infty} \frac{1}{d} \sum_{k=1}^d \nu P^{(n + k)} = \pilimn→∞​d1​∑k=1d​νP(n+k)=π. Therefore, the limit and the stationary distribution are not the same.

Therefore, we formally have the following theorem:

If PPP is irreducible and aperiodic, then there exists a unique stationary distribution π\piπ such that lim⁡n→∞νP(n)=π\lim_{n \to \infty} \nu P^{(n)} = \pilimn→∞​νP(n)=π with initial probability distribution ν\nuν.

Countable MC convergence to stationary distribution​

If MC is countable, irreducible and aperiodic, the positive recurrent states is similar to finite MC. lim⁡n→∞P(n)=π\lim_{n \to \infty} P^{(n)} = \pilimn→∞​P(n)=π.

  • A irreducible, aperiodic and positive recurrent MC called ergodic.

Furthermore, If a stationary distribution exists, then the chain is positive recurrent.

Success-run chain​

For a MC with transition probability px,x+1=px,px,0=qxp_{x,x+1} = p_x, p_{x,0} = q_{x}px,x+1​=px​,px,0​=qx​ where px+qx=1p_{x} + q_{x} = 1px​+qx​=1, we call it a success-run chain. It can be shown that XnX_nXn​ is:

  • Transient if lim⁡n→∞∏j=0n−1pj>0\lim_{n \to \infty} \prod_{j = 0}^{n-1} p_{j} > 0limn→∞​∏j=0n−1​pj​>0.
  • Null recurrent if lim⁡n→∞∏j=0n−1pj=0\lim_{n \to \infty} \prod_{j = 0}^{n-1} p_{j} = 0limn→∞​∏j=0n−1​pj​=0.
  • Positive recurrent if ∑n→∞∏j=0n−1pj<∞\sum_{n \to \infty} \prod_{j = 0}^{n-1} p_{j} < \infty∑n→∞​∏j=0n−1​pj​<∞.
