Markov Chain

Most of the markov chain property and defintion is the same as Markov Chain. And those difference list in this page.

When we say MC is infinite, we mean the state space is infinite.

Markov Chain State​

Let state i,ji,ji,j, we define i→ji \to ji→j iff jjj is accessible from state iii. Addtionally, we define i↔ji \leftrightarrow ji↔j iff i→ji \to ji→j and j→ij \to ij→i, and we say iii and jjj can communicate to each other.

  • such ↔\leftrightarrow↔ is called equivalence equation which a equation is symmetric, relective and transitive.
  • ∃k,l∈S,lim⁡n→∞pkl(n)=0  ⟹  ∀i,j∈S,s.t.k→i,j→l,lim⁡n→∞pij(n)=0\exists k, l\in S,\lim_{n \to \infty} p_{kl}^{(n)} = 0 \implies \forall i,j \in S, s.t. k\to i, j\to l, \lim_{n \to \infty} p_{ij}^{(n)} = 0∃k,l∈S,limn→∞​pkl(n)​=0⟹∀i,j∈S,s.t.k→i,j→l,limn→∞​pij(n)​=0.

We define a communication class as a set of states that can communicate to each other. if there exist only one communication class, then the markov chain is called irreducible. Otherwise, it is called reducible.

Periodicity​

Firstly, let's define Ji={n≥1:pii(n)>0}J_i = \{n\ge 1: p_{ii}^{(n)} > 0\}Ji​={n≥1:pii(n)​>0} which means the set of all possible time that visit iii. Then we have di=gcd⁡{Ji}d_i = \gcd\{J_i\}di​=gcd{Ji​} as the period.

  • if di=1d_i = 1di​=1, then iii is aperiodic.
  • else iii is periodic.
  • all states in the same communication class share the same periodicity.

Positive and Null Recurrence​

For infinite MC, we have two different types of recurrence:

  • We call a chain is null recurrent iff it's recurrent and lim⁡n→∞pxy(n)=0\lim_{n\to\infty} p_{xy}^{(n)} = 0limn→∞​pxy(n)​=0 for all x,y∈Sx,y \in Sx,y∈S.
  • Else it's positive recurrent.

Recall τi=min⁡{n≥1:Xn=i}\tau_i = \min\{n \geq 1: X_n = i\}τi​=min{n≥1:Xn​=i} which means the time that first time visit state iii. Then we have:

  • for a positive recurrent chain, E[τi]<∞E[\tau_i] < \inftyE[τi​]<∞ for all i∈Si \in Si∈S.
  • for a null recurrent chain, E[τi]=∞E[\tau_i] = \inftyE[τi​]=∞ but P(τi<∞)=1P(\tau_i < \infty) = 1P(τi​<∞)=1 for all i∈Si \in Si∈S.
  • for a transient chain, P(τi=∞)>0P(\tau_i = \infty) > 0P(τi​=∞)>0 for all i∈Si \in Si∈S.

If a chain is not positive recurrent, then it's either null recurrent or transient.
