Variational Inference

Recall the posterior distribution p(z∣x)=p(x,z)p(x)p(z|x) = \frac{p(x,z)}{p(x)}p(z∣x)=p(x)p(x,z)​ is the distribution of the latent variables given the observed data where p(x)=∫p(x,z)dzp(x) = \int p(x,z) dzp(x)=∫p(x,z)dz is the marginal distribution of the observed data. But generally, when we face high dimensional latent variables, it becomes intractable to compute the posterior distribution. Specifically, we have the following problem:

  • computing a posterior distribution p(z∣x)p(z|x)p(z∣x) is intractable
  • computing the evidence/likelihood p(x)p(x)p(x) is intractable
  • computing marginal distributions p(z)p(z)p(z) is intractable
  • sampling from the posterior distribution p(z∣x)p(z|x)p(z∣x) is intractable

Variational Inference is an approximate inference method that allows us to approximate the posterior distribution p(z∣x)p(z|x)p(z∣x) by a simpler distribution q(z)q(z)q(z). It works follow:

  1. Choose a tractable distribution q(z)∈Qq(z) \in Qq(z)∈Q from a feasible set QQQ to approximate the posterior distribution p(z∣x)p(z|x)p(z∣x).
    • q(z)=N(z∣μ,Σ)q(z) = \mathcal{N}(z| \mu, \Sigma)q(z)=N(z∣μ,Σ) sometimes a good choice
  2. Encode some notion of difference between p(z∣x)p(z|x)p(z∣x) and qqq that can be efficiently estimated.
    • KL divergence is a good choice
  3. Minimize the difference
    • usually by iterative optimization

KL Divergence​

The KL (Kullback-Leibler) divergence is a measure of the difference between two probability distributions ppp and qqq. It is defined as:KL(q(z)∣∣p(z∣x))=∫q(z)log⁡q(z)p(z∣x)dz=Ez∼qlog⁡q(z)p(z∣x)KL(q(z)||p(z|x)) = \int q(z) \log \frac{q(z)}{p(z|x)} dz = \mathbb{E}_{z\sim q} \log \frac{q(z)}{p(z|x)}KL(q(z)∣∣p(z∣x))=∫q(z)logp(z∣x)q(z)​dz=Ez∼q​logp(z∣x)q(z)​

Some properties of KL divergence:

  • KL(q∣∣p)≥0KL(q||p) \geq 0KL(q∣∣p)≥0
  • KL(q∣∣p)=0KL(q||p) = 0KL(q∣∣p)=0 if and only if q=pq = pq=p
  • KL(q∣∣p)≠KL(p∣∣q)KL(q||p)\neq KL(p||q)KL(q∣∣p)=KL(p∣∣q)
  • KL divergence is not symmetric so that it is not a metric

Information Projection​

I-projection: q∗=arg⁡min⁡q∈QKL(q∣∣p)=Ex∼qlog⁡q(x)p(x)q^* = \arg \min_{q\in Q} KL(q||p) = \mathbb{E}_{x\sim q} \log \frac{q(x)}{p(x)}q∗=argminq∈Q​KL(q∣∣p)=Ex∼q​logp(x)q(x)​. It follows:

  • p≈q  ⟹  KL(q∣∣p)≈0p\approx q \implies KL(q||p) \approx 0p≈q⟹KL(q∣∣p)≈0
  • I-projection underestimates support, and does not yield the correct moments.
  • KL(q∣∣p)KL(q||p)KL(q∣∣p) penalizes qqq having mass where ppp has none.

Moment Projection​

M-projection: q∗=arg⁡min⁡q∈QKL(q∣∣p)=Ex∼qlog⁡p(x)q(x)q^* = \arg \min_{q\in Q} KL(q||p) = \mathbb{E}_{x\sim q} \log \frac{p(x)}{q(x)}q∗=argminq∈Q​KL(q∣∣p)=Ex∼q​logq(x)p(x)​. It follows:

  • p≈q  ⟹  KL(p∣∣q)≈0p\approx q \implies KL(p||q) \approx 0p≈q⟹KL(p∣∣q)≈0
  • KL(p∣∣q)KL(p||q)KL(p∣∣q) penalizes qqq having mass where ppp has none.
  • M-projection yields a distribution q(x)q(x)q(x) with the correct mean and covariance.

ELBO: Evidence Lower Bound​

Sometimes evaluating the KL divergence is intractable due to the integral over zzz and the term p(z∣x)p(z|x)p(z∣x) is intractable to normalize. That is, we would like maximize the ELBO (which is the same as minimizing the KL divergence).

KL(q(z)∣∣p(z∣x))=Ez∼qlog⁡q(z)p(z∣x)=Ez∼q[log⁡(q(z)p(x)p(z∣x))]=Ez∼qlog⁡[q(z)p(z∣x)]+Ez∼qlog⁡p(x)KL(q(z)||p(z|x)) = \mathbb{E}_{z\sim q} \log \frac{q(z)}{p(z|x)} = \mathbb{E}_{z\sim q} [\log (\frac{q(z) p(x)}{p(z|x)})] = \mathbb{E}_{z\sim q} \log [\frac{q(z)}{p(z|x)}] + \mathbb{E}_{z\sim q} \log p(x)KL(q(z)∣∣p(z∣x))=Ez∼q​logp(z∣x)q(z)​=Ez∼q​[log(p(z∣x)q(z)p(x)​)]=Ez∼q​log[p(z∣x)q(z)​]+Ez∼q​logp(x)

We denote ELBO L(ϕ)=Ez∼qlog⁡[p(z∣x)q(z)]\mathcal{L}(\phi) = \mathbb{E}_{z\sim q} \log [\frac{p(z|x)}{q(z)}]L(ϕ)=Ez∼q​log[q(z)p(z∣x)​], that is, KL(q(z)∣∣p(z∣x))=−L(ϕ)+log⁡p(x)KL(q(z)||p(z|x)) = -\mathcal{L}(\phi) + \log p(x)KL(q(z)∣∣p(z∣x))=−L(ϕ)+logp(x).

Since KL(q(z)∣∣p(z∣x))≥0KL(q(z)||p(z|x)) \geq 0KL(q(z)∣∣p(z∣x))≥0, then L(ϕ)≤log⁡p(x)\mathcal{L}(\phi) \leq \log p(x)L(ϕ)≤logp(x).

Since it's a min/max problem, we would like to use SGD/GD to solve it. If zzz satisfies some distribution, then we can use sampling method to solve the optimization problem.

Interpretaion​

We define entropy as H(p)=−Ex∼plog⁡p(x)H(p) = -\mathbb{E}_{x\sim p} \log p(x)H(p)=−Ex∼p​logp(x) to measure the uncertainty of a distribution ppp.

That is, we have the optimization problem: maximize H(p)H(p)H(p) subject to Ex∼p(x)[fi(x)]=ti\mathbb{E}_{x\sim p(x)}[f_i(x)] = t_iEx∼p(x)​[fi​(x)]=ti​ for i=1,2,⋯ ,ki=1,2,\cdots,ki=1,2,⋯,k.

Exponential Family yields Maximum Entropy​

We have the theorem that Exponential Family of distributions maximize the entropy over all distributions satisfying Ex∼p(x)[fi(x)]=ti\mathbb{E}_{x\sim p(x)}[f_i(x)] = t_iEx∼p(x)​[fi​(x)]=ti​ for i=1,2,⋯ ,ki=1,2,\cdots,ki=1,2,⋯,k.

That is, if QQQ is set of exponential family, then the expected sufficient statistics w.r.t q∗(x)q^*(x)q∗(x) is the same as that w.r.t p(x)p(x)p(x) in M-projection. Since required the p(x)p(x)p(x), then M-projection is intractable. Then most variational inference methods use I-projection.

Example on MRF​

We have the following MRF: p(x∣θ)=exp⁡{∑c∈Cϕc(xc)−log⁡Z(θ)}p(x|\theta) = \exp \{\sum_{c\in C} \phi_c(x_c) - \log Z(\theta)\}p(x∣θ)=exp{∑c∈C​ϕc​(xc​)−logZ(θ)}

We use I-projection to approximate the posterior distribution p(x∣θ)p(x|\theta)p(x∣θ) by a simpler distribution q(x)q(x)q(x). Then we have the following optimization problem:

q∗=arg⁡min⁡q∈QKL(q∣∣p)=Ex∼qlog⁡q(x)p(x∣θ)=arg⁡min⁡q∈QEx∼q[log⁡q(x)−∑c∈Cϕc(xc)+log⁡Z(θ)]=arg⁡max⁡q∈QH(q)+∑c∈CEq[ϕc(xc)]q^* = \arg \min_{q\in Q} KL(q||p) = \mathbb{E}_{x\sim q} \log \frac{q(x)}{p(x|\theta)} \\= \arg \min_{q\in Q} \mathbb{E}_{x\sim q} [\log q(x) - \sum_{c\in C} \phi_c(x_c) + \log Z(\theta)] \\= \arg \max_{q\in Q} H(q) + \sum_{c\in C}\mathbb{E}_{q}[\phi_c(x_c)]q∗=argminq∈Q​KL(q∣∣p)=Ex∼q​logp(x∣θ)q(x)​=argminq∈Q​Ex∼q​[logq(x)−∑c∈C​ϕc​(xc​)+logZ(θ)]=argmaxq∈Q​H(q)+∑c∈C​Eq​[ϕc​(xc​)]

Ideally, if p∈Qp\in Qp∈Q, then q∗=pq^* = pq∗=p. But not always happens so we use mean-field approach.

Mean-Field Approach​

We firstly assume q(x)=∏i∈Vqi(xi)q(x) = \prod_{i\in V} q_i(x_i)q(x)=∏i∈V​qi​(xi​), the set QQQ is composed of those distributions that factor out.

Using I-projection above, we have q∗=arg⁡max⁡q∈QH(q)+∑c∈C∑xcq(xc)ϕc(xc)q* = \arg \max_{q\in Q} H(q) + \sum_{c\in C}\sum_{x_c}q(x_c)\phi_c(x_c)q∗=argmaxq∈Q​H(q)+∑c∈C​∑xc​​q(xc​)ϕc​(xc​)

Since H(q)=−Ex∼qlog⁡q(x)=−∑xq(x)[∑ilog⁡qi(xi)]=−∑i∑x[log⁡qi(xi)qi(xi)]q(x)qi(xi)=−∑i∑xi[log⁡qi(xi)qi(xi)]∑x∖xiq(x)qi(xi)=−∑i∑xi[log⁡qi(xi)qi(xi)]=∑iH(qi)H(q) = -\mathbb{E}_{x\sim q} \log q(x) \\= -\sum_{x}q(x)[\sum_i \log q_i(x_i)] \\= -\sum_i\sum_x[\log q_i(x_i)q_i(x_i)]\frac{q(x)}{q_i(x_i)} \\= -\sum_i\sum_{x_i}[\log q_i(x_i)q_i(x_i)]\sum_{x\setminus x_i}\frac{q(x)}{q_i(x_i)} \\= -\sum_i\sum_{x_i}[\log q_i(x_i)q_i(x_i)] \\= \sum_i H(q_i) H(q)=−Ex∼q​logq(x)=−∑x​q(x)[∑i​logqi​(xi​)]=−∑i​∑x​[logqi​(xi​)qi​(xi​)]qi​(xi​)q(x)​=−∑i​∑xi​​[logqi​(xi​)qi​(xi​)]∑x∖xi​​qi​(xi​)q(x)​=−∑i​∑xi​​[logqi​(xi​)qi​(xi​)]=∑i​H(qi​), then we have q∗=arg⁡max⁡q∈Q∑iH(qi)+∑c∈C∑xcq(xc)ϕc(xc)q^* = \arg \max_{q\in Q} \sum_i H(q_i) + \sum_{c\in C}\sum_{x_c}q(x_c)\phi_c(x_c)q∗=argmaxq∈Q​∑i​H(qi​)+∑c∈C​∑xc​​q(xc​)ϕc​(xc​) where ∑xiqi(xi)=1\sum_{x_i}q_i(x_i) = 1∑xi​​qi​(xi​)=1

But it's still not easy to solve. We can use coordinate ascent to solve it.

  1. Initialize {qi(xi)}i∈V\{q_i(x_i)\}_{i\in V}{qi​(xi​)}i∈V​
  2. Iterate over i∈Vi\in Vi∈V
    1. Greedy maximize the objective over qi(xi)q_i(x_i)qi​(xi​) (i.e.) qi(xi)∝exp⁡{∑j∈N(i)∑xjqj(xj)ϕij(xi,xj)}q_i(x_i)\propto \exp \{\sum_{j\in N(i)}\sum_{x_j}q_j(x_j)\phi_{ij}(x_i,x_j)\}qi​(xi​)∝exp{∑j∈N(i)​∑xj​​qj​(xj​)ϕij​(xi​,xj​)} where we can use Lagrangian by setting derivative to zero to solve
    2. Repeat until convergence

This is guaranteed to converge but can only converge to local optima.
