---
title: "Metric"
---

Let $\mathbb{X}$ be arbitrary collection, the function $d: \mathbb{X}\times\mathbb{X}\to \R$ is a **metric** if satisfied:

-   $\forall x,y\in \mathbb{X}, d(x,y) = d(y,x)$
-   $\forall x,y\in \mathbb{X}, d(x,y) = 0 \iff x=y$
-   $\forall x,y,z\in \mathbb{X}, d(x,z) \le d(x,y) + d(y,z)$

That's, for a collection $\mathbb{X}$ and metric $d$ on this collection, $(\mathbb{X}, d)$ is called a metric space.
